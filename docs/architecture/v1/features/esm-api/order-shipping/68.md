# 주문확인 API

> 출처: https://etapi.gmarket.com/68
> 섹션: 주문 · 배송 API
> Tistory post id: `68`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| **[POST] https://sa2.esmplus.com/shipping/v1/Order/OrderCheck/{OrderNo}** |
| 신규 주문(결제완료)에 대해 주문 확인을 처리합니다. 주문확인 시 주문상태가 “배송준비중” 으로 변경되며 고객은 주문 내역에서 동일 상태로 보이게 됩니다. 주문확인 전 주문은 구매자 취소 시 바로 환불되며 주문확인 후 주문은 취소 요청 시 판매자 승인 후 취소됩니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| orderNo | 주문번호 | Y | path | 처리하고자 하는 주문번호 입력 |
| SellerOrderNo | 판매자 자사몰 주문번호 |  | String | - 최대 30byte까지 입력가능 - 판매자몰 주문번호 - 판매자몰 CS와 커뮤니케이션시 이용하는 정보로 자사몰CS에서 확인가능한 실제 번호 연동 |
| SellerItemNo | 판매자 자사몰 상품번호 |  | String | 최대 30byte까지 입력가능 G마켓/옥션 주문번호와 매칭되는 판매자 자사몰의 상품번호 판매자 자사몰의 CS와 커뮤니케이션 시 이용하는 정보로 자사몰CS에서 확인가능한 실제 번호 연동 |

```
JSON
1. 판매자 자사몰 정보 미입력시 url 내 주문번호 입력하여 호출
2. 판매자 자사몰 정보 입력시 url 및 아래 전문 내 정보 입력하여 호출
{
“SellerOrderNo”: “string”,
“SellerItemNo”: “string”
}
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
|  |  |  |  |
| --- | --- | --- | --- |
| ResultCode | 결과 | string | 성공: Success 실패: Error Massage 참고 |
| Message | 메시지 | string |  |
| Data > IsChanged | 정보수정여부 | int | - 주문확인 전 구매자가 수령지(주소)변경 했는지 여부 - 변경시 주문 조회(orderStatus > 2) 로 변경 정보 조회하여 주문정보 갱신 필요 1: 변경됨 0: 변경없음 |

}
}

### **Error Code**

|  |  |  |
| --- | --- | --- |
| **결과코드** | **메시지** | **원인** |
| 3000 | 취소된 주문번호입니다. | 취소된 상태의 주문상태 |
| 3000 | 주문확인 할 수 없는 상태입니다. | 주문확인 할 수 없는 상태의 주문상태 |
| 3000 | 이미 주문확인 처리된 건입니다. | 주문확인 할 수 없는 상태의 주문상태 |
| 3000 | 해당 주문 내역이 없습니다. | 토큰의 ID가 아닌 판매자의 주문건이거나 주문번호 오류 |
| 3000 | 매개 변수 값을 Int64에서 Int32(으)로 변환하지 못했습니다. | OrderNo 자릿수 오류 또는 토큰 오 사용 |
| 3000 | 해당 주문은 3PL 주문입니다 | 풀필먼트 스타배송 상품의 주문 건을 처리하려고 할 경우 |
| 3000 | 해당 주문은 선물수락대기 상태로 주문확인이 불가합니다. | giftOrderStatus: 1(선물수락대기) 상태에서 주문확인 처리 시도할 경우 |
| 2141 | 제휴사 주문번호의 최대길이는 30byte입니다.\\n | 판매자 자사몰 주문번호가 30byte 초과했을 경우 |
| 2141 | 제휴사 상품번호의 최대길이는 30byte입니다.\\n | 판매자 자사몰 상품번호가 30byte 초과했을 경우 |
