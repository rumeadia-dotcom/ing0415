# 주문조회 API

> 출처: https://etapi.gmarket.com/67
> 섹션: 주문 · 배송 API
> Tistory post id: `67`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| **Description** |
| **[POST]** https://sa2.esmplus.com/shipping/v1/Order/RequestOrders |
| 결제 완료된 주문을 조회합니다. 이후 주문 상태별 조회 가능하면 클레임(취소, 반품, 교환, 미수령신고) 주문은 조회되지 않습니다. 선물 주문 조회시 선물 수락 전후 ‘주문 옵션’ 정보가 변경될 수 있습니다. (ex. 선물 수락 전 : A 옵션 , 선물 수락 후 : B 옵션) **주문조회는 5초당 1회 호출 가능합니다. (단, 주문번호로 조회하는 경우 제한 없습니다.)** |
