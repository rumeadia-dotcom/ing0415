# 재고 수정 API

> 출처: https://etapi.gmarket.com/194
> 섹션: 상품 API
> Tistory post id: `194`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| 수정 : [PUT] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/stock |
| 상품 재고 정보를 수정하는 API입니다. 본품만 판매하는 상품만 수정 가능하며 옵션 사용 상품 및 풀필먼트 스타배송 상품은 재고 수정 불가 합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 항목 | 항목명 | 필수여부 | Type | Description |
| goodsNo | 마스터 상품번호 | Y | path | 변경하고자 하는 마스터 상품번호 입력 |
| stock > gmkt | 지마켓 재고 |  | int | G마켓 판매 재고 입력 1 ~ 99,999 내 입력 가능 |
| stock > iac | 옥션 재고 |  | int | G마켓 판매 재고 입력 1 ~ 99,999 내 입력 가능 |

```
JSON
수정시
{
  "stock": {
    "gmkt": int,
    "iac": int
  }
}
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| 항목 | 항목명 | Type | Description |
| resultCode | 결과코드 | int |  |
| message | 메시지 | string |  |
| goodsNo | 마스터 상품번호 | int |  |
| data | 데이터 | object | 실패시 내려가는 정보 |

#### JSON\_Success

```
{
    "resultCode": 0,
    "message": "string",
    "goodsNo": int
}
```

#### JSON\_Fail

```
{
    "resultCode": 1000,
    "message": "string",
    "data": null
}
```

#### Sample Code\_Success

```
{
    "resultCode": 0,
    "message": "[IAC] 성공 / [GMKT] 성공",
    "goodsNo": 2303461921
}
```

#### Sample Code\_Fail

```
{
    "resultCode": 1000,
    "message": "[GMKT] 재고는 99999개 이하이어야 합니다.",
    "data": null
}
```
