> 섹션: 상품 API
> Tistory post id: `4`
> 추출 시점: 2026-05-29

|  |
| --- |
| Description |
| 대분류 전체 조회 [GET] https://sa2.esmplus.com/item/v1/categories/site-cats  하위 카테고리 개별 조회 [GET] https://sa2.esmplus.com/item/v1/categories/site-cats/{siteCatCode} |
| G마켓, 옥션 사이트 카테고리 조회하여 상품 등록시 필수로 입력합니다. (leaf 카테고리로 상품 등록 가능) 사이트별 개별 조회시 해당 사이트 ID 1개만 입력된 인증토큰으로 호출 해주시면 됩니다.  참고 url ) 옥션 카테고리 xml 링크 : https://script.auction.co.kr/category/categories.xml (IsLeaf="true" 경우 상품 등록 가능) |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 항목 | 항목명 | 필수여부 | Type | Description |
| siteCatCode | G마켓/옥션 카테고리코드 | Y | path | 대분류 전체 카테고리 조회시 아래 URL로 호출 [GET] https://sa2.esmplus.com/item/v1/categories/site-cats 조회한 카테고리코드 기준 하위 카테고리 확인 가능 |

```
JSON
URL만 호출 / Request Body 없음
[GET] https://sa2.esmplus.com/item/v1/categories/site-cats/100000002

전체 카테고리 조회 - 사이트별 대분류 조회 가능
[GET] https://sa2.esmplus.com/item/v1/categories/site-cats
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| 항목 | 항목명 | Type | Description |
| resultCode | 결과코드 | int | 0 : 성공 나머지 : 실패 |
| message | 메시지 | string | 성공 or 실패사유 |
| catCode | 카테고리코드 | string | 조회대상 카테고리 코드 subCats Object는 하위카테고리코드 |
| catName | 카테고리명 | string | 조회대상 카테고리명 subCats Object는 하위카테고리명 |
| isLeaf | 최하위 카테고리여부 | boolean | true : 최하위 카테고리인 경우 false : 하위 카테고리가 있을 경우  subCats Object는 하위카테고리의 최하위카테고리 여부 상품 등록/수정 시 최하위 카테고리를 연동해야함 |

#### JSON\_Success

```
{
  "catCode": "string",
  "catName": "string",
  "isLeaf": true,
  "subCats": [
    {
      "catCode": "string",
      "catName": "string",
      "isLeaf": true
    }
  ]
}
```

#### JSON\_Fail

```
{
   "resultCode": 0,
   "message": "string"
}
```
