# 옥션 복수 할인 API

> 출처: https://etapi.gmarket.com/37
> 섹션: 상품 API
> Tistory post id: `37`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| 등록/수정 : [POST] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/customer-benefit/special-discount  해제 : [DELETE] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/customer-benefit/special-discount  조회 : [GET] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/customer-benefit/special-discount |
| 구매수량별 복수구매할인 혜택을 제공할 수 있습니다 (옥션 상품만 가능) **※ 우수회원 할인 설정 2022년 2월 28일 종료** |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| goodsNo | 마스터상품번호 | Y | path |  |
| multiplePurchaseDiscount | 복수구매할인여부 | Y | boolean | true : 사용함 할인 해제 경우 DELETE 로 호출하여 처리 |
| multiplePurchaseDiscountAmount | 복수구매할인금액 | Y | number | 구매수량당 N원 할인(100원 단위 입력) |
| multiplePurchaseDiscountQuantity | 복수구매할인기준 | Y | number | 주문수량 N개 이상(단위설정) 2 ,3, 4, 5 중 선택 |
| startDate | 적용시작일자 | Y | date | YYYY-MM-DD |
| endDate | 적용종료일자 | Y | date | YYYY-MM-DD |

```
JSON
등록/수정
{
  "multiplePurchaseDiscount": true,
  "multiplePurchaseDiscountAmount": 0,
  "multiplePurchaseDiscountQuantity": 0,
  "startDate": "string",
  "endDate": "string"
}

해제 - URL만 호출/Request Body없음
[DELETE] https://sa2.esmplus.com/item/v1/goods/1139546520/customer-benefit/special-discount

조회 - URL만 호출/Request Body없음
[GET] https://sa2.esmplus.com/item/v1/goods/1139546520/customer-benefit/special-discount
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
| goodsNo | 마스터상품번호 | int | 조회 시 내려가는 정보 |
| resultCode | 결과코드 | int | 실패 시 내려가는 정보 |
| message | 메시지 | string | 실패 시 내려가는 정보 |
