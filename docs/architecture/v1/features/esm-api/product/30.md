| resultCode | 결과코드 | int |  |
| message | 메시지 | string | 실패 시 내려가는 정보 |

| innerResultCode | Site피딩관리코드 | string | 마스터 상품 기준으로 각 Site에 등록된 상품번호를 조회 한 정보 |
| innerMessage | Site피딩관리코드메시지 | string | 마스터 상품 기준으로 각 Site에 등록된 상품번호를 조회 한 정보 |

| siteDetail > gmkt > SiteGoodsComment | G마켓메시지 | string | 마스터 상품 기준으로 각 Site에 등록된 상품번호를 조회 한 정보 |
| siteDetail > iac > SiteGoodsNo | 옥션상품번호 | string | 마스터 상품 기준으로 각 Site에 등록된 상품번호를 조회 한 정보 |
| siteDetail > iac > SiteGoodsComment | 옥션메시지 | string | 마스터 상품 기준으로 각 Site에 등록된 상품번호를 조회 한 정보 |
