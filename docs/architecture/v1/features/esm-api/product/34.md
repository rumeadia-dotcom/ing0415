# 사이트 카테고리별 등록 옵션 정보 조회 API

> 출처: https://etapi.gmarket.com/34
> 섹션: 상품 API
> Tistory post id: `34`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| [GET] https://sa2.esmplus.com/item/v1/goods/option-policies?siteId={siteId}&siteCatCode={siteCatCode} |
| G마켓, 옥션 사이트 카테고리별 등록 가능 옵션 및 추가금 정보 조회 가능합니다. (leaf 카테고리로 입력) 상품 동시 등록/수정시 G마켓, 옥션 카테고리별 주문옵션 모두 사용 Y 리턴된 옵션 유형에 대해서만 옵션 등록/수정 가능합니다.  추가금 설정 범위는 판매가 1만원 이상 설정시 가능한 금액 범위이며 1만원 미만 판매가 상품 경우 추가금 설정 가능한 카테고리 경우 아래 기준 공통 정책 적용 됩니다. - 2,000원 ~ 10,000원 : 판매가 기준  -50 ~ 100% 까지 - 2,000원 미만 : 판매가 기준 0 ~ 100% 까지 |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| siteId | 사이트 구분 | Y | path | 1 : 옥션 2 : 지마켓 |
| siteCatCode | 사이트 카테고리코드 | Y | path | 상품 등록 최종 카테고리 기준으로 조회 가능 |

```
JSON
URL만 호출 / Request Body 없음
[GET] https://sa2.esmplus.com/item/v1/goods/option-policies?siteId=1&siteCatCode=23141600
```

### **Response Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
| optionUseYn | 옵션 사용 여부 |  | string | Y : 사용 가능 N : 사용 불가 |
| customOptionUseYn | 선택형(2, 3개 조합) 자율 옵션 사용 가능 여부 |  | string | Y : 사용 가능 N : 사용 불가 |
| optionType > genrlUseYn | 선택형 가능 여부 |  | string | Y : 사용 가능 N : 사용 불가 |
| optionType > twoCmbtUseYn | 2개 조합형 가능 여부 |  | string | Y : 사용 가능 N : 사용 불가 |
| optionType > threeCmbtUseYn | 3개 조합형 가능 여부 |  | string | Y : 사용 가능 N : 사용 불가 |
| optionType > textUseYn | 택스트형 가능 여부 |  | string | Y : 사용 가능 N : 사용 불가 |
| optionType > calcUseYn | 계산형 가능 여부 |  | string | Y : 사용 가능 N : 사용 불가 |
| addAmnt > addAmntUseYn | 추가금 사용 가능 여부 |  | string | Y : 사용 가능 N : 사용 불가 |
| addAmnt > minRate | 추가금 최소 설정 범위 |  | int |  |
| addAmnt > maxRate | 추가금 최대 금액 범위 |  | int |  |
| customTextOptionUseYn | 텍스트형 자율 옵션 사용 가능 여부 |  | string | Y : 사용 가능 N : 사용 불가 |
