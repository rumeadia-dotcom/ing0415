# 추천옵션 항목 조회 API

---

|  |
| --- |
| Description |
| 카테고리별 추천옵션 조회 : [GET] https://sa2.esmplus.com/item/v1/options/recommended-opts?catCode={siteCatCode} 추천옵션별 선택항목 조회 : [GET] https://sa2.esmplus.com/item/v1/options/recommended-opts/{recommendedOptNo} |
| 카테고리별 추천옵션 조회시 recommendedOptNo = 0 이 포함된 경우 해당 카테고리는 직접입력 옵션 허용됩니다. 선택형(2, 3개 조합 포함), 텍스트형 직접입력 허용 여부가 카테고리별로 다르니 등록 가능한 옵션 유형 확인하시길 바랍니다.  카테고리별 추천옵션 조회시 조회하는 사이트 ID 만 포함된 인증토큰으로 조회하셔야 합니다. (지마켓 카테고리 조회시 지마켓 ID 만 포함된 인증토큰으로, 옥션 카테고리 조회시 옥션 ID 만 포함된 인증토큰으로) |

**카테고리별 추천옵션 조회**

| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
| siteCatCode | 사이트 카테고리 코드 | Y | path |  |

**추천옵션별 선택 항목 조회**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
| recommendedOptNo | 추천옵션코드 | Y | path | ※ 0 : 직접입력 경우 조회 항목 없음 |

