# 카테고리별 상품명 수정 가능 여부 조회 API

> 출처: https://etapi.gmarket.com/151
> 섹션: 상품 API
> Tistory post id: `151`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| **[GET] https://sa2.esmplus.com/item/v1/goods/goods-name-policies?siteId={siteId}&siteCatCode={siteCatCode}** |
| G마켓, 옥션 사이트 카테고리별 상품명 수정 가능 여부 조회 가능합니다. (leaf 카테고리로 입력) 상품 동시 등록/수정시 G마켓, 옥션 카테고리별 모두 사용 Y 리턴된 경우 상품명 수정 가능 합니다. 카테고리별 정책 외 상품 등록 후 10일 미만이거나 10일 내 판매 이력 없으며 수정 가능 합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| siteId | 사이트 구분 | Y | path | 1 : 옥션 2 : 지마켓 |
| siteCatCode | 사이트 카테고리코드 | Y | path | 상품 등록 최종 카테고리 기준으로 조회 가능 |

```
JSON
URL만 호출 / Request Body 없음
[GET] https://sa2.esmplus.com/item/v1/goods/goods-name-policies?siteId=1&siteCatCode=55020600
```
