# 상세설명 수정 API

> 출처: https://etapi.gmarket.com/24
> 섹션: 상품 API
> Tistory post id: `24`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| [POST] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/descriptions |
| 상품 상세설명 html 수정하는 API 입니다. 영문/중문/일문을 입력하면 G마켓 글로벌 사이트에서 언어별 상세설명이 노출 됩니다. |

### 

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
| goodsNo | 마스터 상품번호 | Y | path | 변경하고자 하는 마스터 상품번호 입력 |
| descNew | (국문) 상세설명 | Y | stirng | 상품상세 Html 입력 iframe, Script 및 max-width 등은 사용 불가 |
| descNewEng | (영문) 상세설명 |  | string |  |
| descNewChn | (중문) 상세설명 |  | string |  |
| descNewJpn | (일문) 상세설명 |  | string |  |

```
JSON
{
  "descNew": "string",
  "descNewEng": "string",
  "descNewChn": "string",
  "descNewJpn": "string"
}

Sample
{
  "descNew": "<div class=\"ee-contents\"><p class=\"\">국문상세설명</p><p><br></p><p><br></p></div>",
  "descNewEng": "<div class=\"ee-contents\"><p class=\"\">영문상세설명</p><p><br></p><p><br></p></div>",
  "descNewChn": "<div class=\"ee-contents\"><p class=\"\">중문상세설명</p><p><br></p><p><br></p></div>",
  "descNewJpn": "<div class=\"ee-contents\"><p class=\"\">일문상세설명</p><p><br></p><p><br></p></div>"
}
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
|  |  |  |  |
| --- | --- | --- | --- |
| goodsNo | 마스터상품번호 | int | 조회 시 내려가는 정보 |
| resultCode | 결과코드 | int | 실패 시 내려가는 정보 |
| message | 메시지 | string | 실패 시 내려가는 정보 |
| data | 데이터 | object | 실패 시 내려가는 정보 |

#### JSON\_Success

```
{
  "goodsNo": 0,
  "resultCode": 0,
  "message": null  
}
```

#### JSON\_Fail

```
{
   "resultCode": 0,
   "message": "string"
}
```

#### Sample Code\_Success

```
{
    "goodsNo": 2398849144,
    "resultCode": 0,
    "message": null
}
```

#### Sample Code\_Fail

```
{
  "resultCode":1000,
  "message":"국문 상품상세는 필수 입니다."
}
```

### **Error Code**

|  |  |  |
| --- | --- | --- |
| **결과코드** | **메시지** | **원인** |
|  |  |  |
| --- | --- | --- |
| 1000 | 국문 상품상세는 필수 입니다. | 필수 값 입력하지 않았을 경우 |
