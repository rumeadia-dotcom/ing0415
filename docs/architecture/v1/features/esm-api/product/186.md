# 가격 수정 API

> 출처: https://etapi.gmarket.com/186
> 섹션: 상품 API
> Tistory post id: `186`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| 수정 : [PUT] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/price 조회 : [GET] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/price |
| 상품 가격만 수정하는 API입니다. 상품 판매상태가 판매중인 경우만 수정 되고 판매중지 상품은 가격 수정되지 않습니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 항목 | 항목명 | 필수여부 | Type | Description |
| goodsNo | 마스터 상품번호 | Y | path | 변경하고자 하는 마스터 상품번호 입력 |
| gmkt | 지마켓 가격 |  | number | G마켓에서 판매될 금액 입력(십억이상 입력 불가) 10원단위로 등록 |
| iac | 옥션 가격 |  | number | 옥션에서 판매될 금액 입력(십억이상 입력 불가) 10원단위로 등록 |

```
JSON
수정시
{
  "gmkt": int,
  "iac": int
}

조회시
https://sa2.esmplus.com/item/v1/goods/{goodsNo}/price
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| 항목 | 항목명 | Type | Description |
| gmkt | 지마켓 가격 | number |  |
| iac | 옥션 가격 | number |  |
| resultCode | 결과코드 | int |  |
| message | 메시지 | string |  |
| goodsNo | 마스터 상품번호 | int |  |
| data | 데이터 | object | 실패시 내려가는 정보 |

#### JSON\_Success

```
{
  "resultCode": 0,
  "message": "string",
  "goodsNo": 0
}
```

#### JSON\_Fail

```
{
  "ResultCode": "Success",
  "Message": "string",
  "Data": {}
}
```

#### Sample Code\_Success

```
{
    "resultCode": 0,
    "message": "[GMKT] 성공",
    "goodsNo": 4805864401
}
```
