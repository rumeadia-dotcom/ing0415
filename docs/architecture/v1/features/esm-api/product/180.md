# 이벤트 홍보별 상품 등록/수정/삭제 API

> 출처: https://etapi.gmarket.com/180
> 섹션: 상품 API
> Tistory post id: `180`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| 등록/수정: [POST] https://sa2.esmplus.com/item/v1/event-promotions/{promotionNo}/goods  조회: [GET] https://sa2.esmplus.com/item/v1/event-promotions/{promotionNo}/goods  삭제 : [DELETE] https://sa2.esmplus.com/item/v1/event-promotions/{promotionNo}/goods  상품기준 조회: [GET] https://sa2.esmplus.com/item/v1/event-promotions/goods/{siteGoodsNo} |
| 등록된 이벤트 홍보별 노출 상품을 등록할 수 있습니다. 이벤트 홍보별 등록 가능한 상품수는 최대 1000개 입니다. 상품을 등록시에는 옥션 상품만, 지마켓 상품만 각각 개별로 등록 가능 합니다. 지마켓 상품 등록시 지마켓 ID 만 포함된 인증티켓으로, 옥션 상품 등록시 옥션 ID 만 포함된 인증티켓으로 등록 가능합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 항목 | 항목명 | 필수여부 | Type | Description |
| siteGoodsNos | 사이트 상품번호 | Y | string | 최대 1000개까지 등록/수정 가능 |

```
JSON
상품 등록/수정
{
  "siteGoodsNos": [
    "string"
  ]
}

조회 [GET] https://sa2.esmplus.com/item/v1/event-promotions/1/goods
삭제 [DELETE] https://sa2.esmplus.com/item/v1/event-promotions/1/goods
상품별 조회 [GET] https://sa2.esmplus.com/item/v1/event-promotions/goods/E123456789
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| 항목 | 항목명 | Type | Description |
| promotionNo | 이벤트 홍보 번호 | int |  |
| siteGoodsNos | 사이트 상품번호 | string |  |

#### JSON\_Success

```
{
    "promotionNo": int,
    "siteGoodsNos": [
        "siteGoodsNo",
        "siteGoodsNo"
    ]
}
```

#### JSON\_Fail

```
{
  "ResultCode": "Success",
  "Message": "string",
  "Data": {}
}
```

#### Sample Code\_Success

```
등록/수정시
{
    "promotionNo": 7
}

조회시
{
    "promotionNo": 7,
    "siteGoodsNos": [
        "E281479617"
    ]
}
```

#### Sample Code\_Fail

```
{
    "resultCode": 1000,
    "message": "API 호출 시 오류가 발생 했습니다. ID는 하나만 입력 해야합니다.",
    "data": null
}
```
