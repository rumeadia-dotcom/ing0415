> 섹션: 상품 API
> Tistory post id: `161`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| 고시 상품군 조회: [GET] https://sa2.esmplus.com/item/v1/official-notice/groups 상품군별 리스트 조회 : [GET] https://sa2.esmplus.com/item/v1/official-notice/groups/{officialNoticeNo}/codes |
| 고시 상품군 및 리스트 조회하는 API 입니다. 상품 등록/수정시 해당되는 상품군 및 상품군 내 리스트 확인 후 정확한 정보 입력 해주시면 됩니다. |

### **Request Description**

**상품군별 리스트 조회**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 항목 | 항목명 | 필수여부 | Type | Description |
| officialNoticeNo | 상품군 코드 | Y | path | 1 ~ 41 내 조회 가능 |

```
JSON
고시 상품군 조회 - URL만 호출 / Request Body 없음
[GET] https://sa2.esmplus.com/item/v1/official-notice/groups

상품군별 리스트 조회 - URL만 호출 / Request Body 없음
[GET] https://sa2.esmplus.com/item/v1/official-notice/groups/{officialNoticeNo}/codes
```

### **Response Description**

**고시 상품군 조회**

|  |  |  |  |
| --- | --- | --- | --- |
| 항목 | 항목명 | Type | Description |
| officialNoticeNo | 상품군 코드 | number |  |
| officialNoticeName | 상품군명 | string |  |

**상품군 리스트 조회**

|  |  |  |  |
| --- | --- | --- | --- |
| 항목 | 항목명 | Type | Description |
| officialNoticeItemelementCode | 고시 상세 코드 | string |  |
| officialNoticeItemelementName | 고시 상세명 | string |  |
| guideText | 가이드 | string |  |
| isExtraMark | 필수 여부 | boolean |  |

#### JSON\_Success

```
고시 상품군 조회
{
    "groups": [
        {
            "officialNoticeNo": number,
            "officialNoticeName": "string"
        }
    ]
}

상품군 리스트 조회
{
    "codes": [
        {
            "officialNoticeItemelementCode": "string",
            "officialNoticeItemelementName": "string",
            "guideText": "string",
            "isExtraMark": boolean
        }
    ]
}
```

#### JSON\_Fail

```
{
   "resultCode": 0,
   "message": "string"
}
```

