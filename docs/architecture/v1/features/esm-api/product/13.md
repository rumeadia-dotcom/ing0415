# ESM-Site 카테고리 매칭조회 API

> 출처: https://etapi.gmarket.com/13
> 섹션: 상품 API
> Tistory post id: `13`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| 전체 사이트 카테고리 조회 [GET] https://sa2.esmplus.com/item/v1/categories/sd-cats/{sdCatCode}/site-cats/full-depth 최하위 사이트 카테고리 조회 [GET] https://sa2.esmplus.com/item/v1/categories/sd-cats/{sdCatCode}/site-cats |
| ESM 카테고리는 등록 가능한 G마켓/옥션 카테고리가 지정되어 있습니다. Site 카테고리와 매칭된 ESM 카테고리를 조회하여 입력해주세요. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 항목 | 항목명 | 필수여부 | Type | Description |
| sdCatCode | ESM 카테고리코드 (최하위 카테고리코드) | Y | path | ESM 카테고리 기준 G마켓/옥션 카테고리가 매칭되어 있음 ESM 카테고리 입력시 매칭 G마켓/옥션 카테고리 입력하여 등록/수정 조회한 카테고리 기준 매칭 카테고리 확인 가능 |

```
URL만 호출 / Request Body 없음
[GET] https://sa2.esmplus.com/item/v1/categories/sd-cats/00320005000000000000/site-cats
[GET] https://sa2.esmplus.com/item/v1/categories/sd-cats/00150003000400000000/site-cats/full-depth
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| 항목 | 항목명 | Type | Description |
| resultCode | 결과코드 | int | 실패 시 내려가는 정보 |
| message | 메시지 | string | 실패 시 내려가는 정보 |
| catCode | 카테고리코드 | string | 조회대상 카테고리 코드 Gmkt Object에 묶여있으면 G마켓 카테고리 IAC Object에 묶여있으면 옥션 카테고리 |
| CatName | 카테고리명 | string | 조회 시 내려가는 정보 |
| sdCategoryTree > IsLeafCategory | 최하위 카테고리여부 | boolean | 조회 시 내려가는 정보 true : 최하위 카테고리 false : 하위 카테고리가 있음 \* 상품 등록/수정시 최하위 카테고리 |
