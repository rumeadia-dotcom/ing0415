# 원산지 리스트 조회 API

---

|  |
| --- |
| **Description** |
| 원산지 리스트 조회 : [GET] https://sa2.esmplus.com/item/v1/origin/codes |
| 상품 등록/수정 시 필요한 원산지 코드 리스트 전체를 조회 합니다. 국내, 해외 지역별 코드로 구분되니 상품에 맞는 원산지 코드 확인 후 등록 하시면 됩니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 항목 | 항목명 | 필수여부 | Type | Description |
|  |  | 없음 |  |  |

```
JSON
원산지 코드 조회 - URL만 호출 / Request Body 없음
[GET] https://sa2.esmplus.com/item/v1/origin/codes
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| 항목 | 항목명 | Type | Description |
| type | 원산지 타입 | number | 1 : 국내 2 : 수입산 3 : 기타 |
| code | 원산지 코드 | int |  |
| lname | 지역 | string | 국내 - 도, 시 해외 - 국가 |
| sname | 지역 상세 | string | 국내 - 시, 군, 구 해외 - 도시 |

#### JSON\_Success

```
{
    "codes": [
        {
            "type": int,
            "code": int,
            "lname": string,
            "mname": string
        }
    ]
}
```
