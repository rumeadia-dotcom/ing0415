> 섹션: 상품 API
> Tistory post id: `31`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| 등록/수정 : [POST] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/minBuyableQuantity 해제 : [DELETE] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/minBuyableQuantity 조회 : [GET] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/minBuyableQuantity |
| 마스터상품 기준으로 상품의 최소구매수량을 설정합니다 제한된 API로 해당 기능은 판매자ID 별도 예외설정 필요합니다 설정없이 사용할 경우 상품수정 시 오류 발생합니다 |

### Request Description

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 항목 | 항목명 | 필수여부 | Type | Description |
| goodsNo | 마스터상품번호 | Y | path |  |
| qty | 최소구매수량 | Y | int |  |

```
JSON
등록/수정
{
  "qty": 0
}

해제, 조회
최소구매수량 해제
[DELETE] https://sa2.esmplus.com/item/v1/goods/1139546520/minBuyableQuantity

최소구매수량 조회
[GET] https://sa2.esmplus.com/item/v1/goods/1139546520/minBuyableQuantity
```
