# 그룹정보조회 API

> 출처: https://etapi.gmarket.com/45
> 섹션: 상품 API
> Tistory post id: `45`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| **Description** |
| 판매자 전체그룹 조회 : [GET] https://sa2.esmplus.com/item/v1/groups 그룹별 조회 : [GET] https://sa2.esmplus.com/item/v1/groups/{groupNo} 그룹상품조회 : [GET] https://sa2.esmplus.com/item/v1/groups/{groupNo}/goods |
| 등록 그룹 및 그룹 내 상품 정보를 조회하는 API입니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| groupNo | 그룹번호 | Y | Path | 적용하고자 하는 그룹번호 입력 |

```
JSON
전체 조회
[GET] https://sa2.esmplus.com/item/v1/groups?offset=1
50개씩 조회할 수 있으며 50개 이상일 때 offset=* 숫자 변경으로 조회 가능
ex. offset=1 (groupNo 1개~50개), offset=2 (groupNo 51개~100개)

그룹별 조회
[GET] https://sa2.esmplus.com/item/v1/groups/784220

그룹상품 조회
[GET] https://sa2.esmplus.com/item/v1/groups/694498/goods
```

### 

### **Error Code**

|  |  |  |
| --- | --- | --- |
| **결과코드** | **메시지** | **원인** |
|  |  |  |
| --- | --- | --- |
| 1000 | 유효하지 않은 그룹 입니다.(해당 그룹 정보가 없습니다.) | 조회하려는 그룹번호가 존재하지 않을 경우 |
| 1000 | 카테고리가 바르지 않습니다.(상품번호) | 카테고리 동일하지 않은 상품 등록할 경우 |
| 1000 | 같은 배송비 상품들로만 그룹이 가능합니다.(상품번호) | 배송비정책 동일하지 않은 상품 등록할 경우 |
| 1000 | 같은 출하지 상품들로만 그룹이 가능합니다. | 출하지번호 동일하지 않은 상품 등록할 경우 |
