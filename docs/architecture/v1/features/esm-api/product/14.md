# 미니샵 카테고리조회 API

> 출처: https://etapi.gmarket.com/14
> 섹션: 상품 API
> Tistory post id: `14`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| [GET] https://sa2.esmplus.com/item/v1/categories/shop-cats/{shopCatCode} [GET] https://sa2.esmplus.com/item/v1/categories/shop-cats/ (전체 조회시) |
| G마켓 / 옥션 판매자 미니샵 카테고리를 상품등록/수정 시 설정할 수 있습니다. ESM에서 G마켓 / 옥션 판매자 미니샵을 설정하시고 조회하시면 코드가 조회 가능합니다. 카테고리 관리는 Plus Shop (유료) 신청한 경우 가능하며 해당 신청은 ESM+ 내 Plus Shop 이용현황에서 신청 가능합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 항목 | 항목명 | 필수여부 | Type | Description |
| shopCatCode | 미니샵 카테고리코드 | Y | path | 조회한 카테고리코드 기준 하위 카테고리 확인 가능 |

```
URL만 호출 / Request Body 없음
[GET] https://sa2.esmplus.com/item/v1/categories/shop-cats/12345678
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| 항목 | 항목명 | Type | Description |
| resultCode | 결과코드 | int | 실패 시 내려가는 정보 |
| message | 메시지 | string | 실패 시 내려가는 정보 |
| subCats > catCode | 카테고리코드 | string | 조회대상 카테고리 코드 Gmkt Object에 묶여있으면 G마켓 카테고리 IAC Object에 묶여있으면 옥션 카테고리 |
| subCats > catName | 카테고리명 | string | 조회 시 내려가는 정보 |
| subCats > isLeaf | 최하위 카테고리 여부 | boolean | 조회 시 내려가는 정보 true : 최하위 카테고리 false : 하위 카테고리가 있음 \* 상품 등록/수정시 최하위 카테고리 |

#### JSON\_Success

```
{
    "subCats": [

        }
```

### **Error Code**

|  |  |  |
| --- | --- | --- |
| 결과코드 | 메시지 | 원인 |
| 1000 | 카테고리 코드 오류 입니다. | 존재하지 않는 카테고리코드 호출했을 경우 |
