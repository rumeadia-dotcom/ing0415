# 상품삭제 API

> 섹션: 상품 API
> Tistory post id: `29`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| **[DELETE] https://sa2.esmplus.com/item/v1/goods/{goodsNo}** |
| 마스터번호 기준으로 상품 삭제하며 동시 등록 상품은 모두 삭제 처리 됩니다. 삭제 처리 전 지마켓, 옥션 상품은 판매중지된 상태 이어야 합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| goodsNo | 마스터 상품번호 | Y | path |  |

```
JSON
URL만 호출 / Request Body 없음
[DELETE] https://sa2.esmplus.com/item/v1/goods/1138007846
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
| goodsNo | 마스터상품번호 | int |  |
| resultCode | 결과코드 | int |  |
| message | 메시지 | string |  |

 JSON\_Success

```
{
    "goodsNo": 0,
    "resultCode": 0,
    "message": null
}
```

#### JSON\_Fail

```
{
   "resultCode": int,
   "message": "string"
}
```

#### Sample Code\_Success

```
{
    "goodsNo": 2375980552,
    "resultCode": 0,
    "message": null
}
```

#### Sample Code\_Fail

```
{
    "resultCode": 1000,
    "message": "상품삭제는 요청된 모든 상품이 [판매중지] 상태일 경우에만 가능합니다."
}

{

    "message": "[IAC] 상품이 없습니다."
```
