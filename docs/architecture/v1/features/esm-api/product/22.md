# 상품명 수정 API

> 출처: https://etapi.gmarket.com/22
> 섹션: 상품 API
> Tistory post id: `22`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| **[PUT] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/goods-name** |
| 상품명을 수정하는 API입니다. 수정 불가한 경우 별도 에러 발생하지 않으며 기존 상품명은 그대로 유지 됩니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
| goodsNo | 마스터 상품번호 | Y | path | 변경하고자 하는 마스터 상품번호 입력 |
| kor | 프로모션명 (공통, 지마켓) | Y | string | - 검색용+프로모션용 최대 100byte 까지 가능 - 일부 카테고리 제외 수정 가능 |
| promotion | 프로모션 상품명 |  | string | - 검색용 + 프로모션용 최대 100byte 까지 입력 가능 - 프로모션용은 상시 수정 가능 - 사이트별 입력시 지마켓 프로모션명 |
| promotionType | 프로모션명 타입 |  | int | 0 : 공통 1 : 사이트별 미입력시 공통 입력으로 처리 |
| promotionIac | 프로모션명 (옥션) |  | string | - 검색용 + 프로모션용 최대 100byte 까지 입력 가능 - 프로모션용은 상시 수정 가능 - 사이트별 입력시 옥션 프로모션명 |

```
JSON
{
  "kor": "string",
  "promotion": "string",
  "promotionType": int,
  "promotionIac": "string",
}

Sample
{
  "kor": "기본상품명 변경",
  "promotion": "프로모션상품명 변경"
}
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
|  |  |  |  |
| --- | --- | --- | --- |
| goodsNo | 마스터상품번호 | int | 조회 시 내려가는 정보 |
| resultCode | 결과코드 | int | 실패 시 내려가는 정보 |
| message | 메시지 | string | 실패 시 내려가는 정보 |
| data | 데이터 | object | 실패 시 내려가는 정보 |

#### JSON\_Success

```
{
  "goodsNo": 0,
  "resultCode": 0,
  "message": null
}
```

#### JSON\_Fail

```
{
   "resultCode": 0,
   "message": "string"
}
```

```

```

#### Sample Code\_Fail

```
}
```

### **Error Code**

|  |  |  |
| --- | --- | --- |
| **결과코드** | **메시지** | **원인** |
| 1000 | [InvalidApiRequestEx][\"검색상품명(kor)을 입력해 주세요.\"] | 검색(기본)상품명 입력하지 않았을 경우 |
