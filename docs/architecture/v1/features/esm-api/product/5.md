> 섹션: 상품 API
> Tistory post id: `5`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| 대분류 전체 조회 [GET] https://sa2.esmplus.com/item/v1/categories/sd-cats/0 하위카테고리 개별 조회 [GET] https://sa2.esmplus.com/item/v1/categories/sd-cats/{sdCatCode} |
| ESM 카테고리 조회하여 상품 등록/수정 시 필수 입력합니다. (상품 등록/수정 시에는 최종 leaf 카테고리로 입력) 조회된 ESM 카테고리 기준으로 매칭된 지마켓, 옥션 카테고리 확인할 수 있습니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 항목 | 항목명 | 필수여부 | Type | Description |
| siteCatCode | ESM 카테고리코드 | Y | path | 조회한 카테고리코드 기준 하위 카테고리 확인 가능 |

```
JSON
URL만 호출 / Request Body 없음
[GET] https://sa2.esmplus.com/item/v1/categories/sd-cats/00320000000000000000

대분류 전체 조회시
[GET] https://sa2.esmplus.com/item/v1/categories/sd-cats/0
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| 항목 | 항목명 | Type | Description |
| resultCode | 결과코드 | int | 실패 시 내려가는 정보 |
| message | 메시지 | string | 실패 시 내려가는 정보 |
| data | 데이터 | object | 실패 시 내려가는 정보 |
| sdCategoryTree > SDCategoryCode | 카테고리코드 | string | 조회 시 내려가는 정보 조회대상 카테고리 코드 |
| sdCategoryTree > SDCategoryName | 카테고리명 | string | 조회 시 내려가는 정보 |
| sdCategoryTree >  IsLeafCategory | 최하위 카테고리여부 | boolean | 조회 시 내려가는 정보 true : 최하위 카테고리 false : 하위 카테고리가 있음  \* 상품 등록/수정 시 최하위 카테고리 |

```
{
```

### **Error Code**

|  |  |  |
| --- | --- | --- |
| 결과코드 | 메시지 | 원인 |
| 1000 | 카테고리코드 확인 후 재호출 해주세요 | 존재하지 않는 카테고리코드 호출했을 경우 |
