| **등록/수정 : [PUT] https://sa2.esmplus.com/item/v1/advertising/listing** |
| 리스트 부가서비스 설정하는 API 입니다. 옥션 프리미엄, 프리미엄플러스 부가 서비스 구매하여 설정 할 수 있습니다. 프리미엄플러스 부가서비스에는 프리미엄이 포함되어 있습니다.  부가서비스 설정은 판매예치금 충전 후 설정 가능 합니다. 사이트별 설정시 예치금 여부에 따라 있는 경우 성공, 없는 경우 실패 처리 됩니다.  G마켓 부가서비스는 25년 9월 15일 종료 되오니 공지 참고 부탁 드립니다. (**[공지 바로가기](https://etapi.gmarket.com/202)**) |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
| goodsNo | 마스터상품번호 | Y | int | 마스터 상품번호 |
| iacPremium > isUse | 옥션프리미엄사용여부 |  | boolean | true : 사용 false : 미사용 |
| iacPremium > applyPeriod | 사용일 |  | int | 7, 15, 30, 60, 90 입력 가능 |
| iacPremium > autoExtendType | 자동연장설정 |  | int | 1 : 자동 연장 미사용 2 : 무제한 연장 3 : 기간 내 연장 |
| iacPremium > autoExtendAmountDay | 자동연장사용일 |  | int | 7, 15, 30, 60, 90 입력 가능 |
| iacPremium > autoExtendEndDate | 제한기간일자 |  | date | - AutoExtendType=3 경우  종료기간 날짜 설정 - YYYY-MM-DD 형식으로 입력 |
| iacPremiumPlus > isUse | 옥션프리미엄플러스사용여부 |  | boolean | true: 사용 false : 미사용 |
| iacPremiumPlus > applyPeriod | 사용일 |  | int | 7, 15, 30, 60, 90 입력 가능 |
| iacPremiumPlus > autoExtendType | 자동연장설정 |  | int | 1 : 자동 연장 미사용 2 : 무제한 연장 3 : 기간 내 연장 |
| iacPremiumPlus > autoExtendAmountDay | 자동연장사용일 |  | int | 7, 15, 30, 60, 90 입력 가능 |
| iacPremiumPlus > autoExtendEndDate | 제한기간일자 |  | date | - AutoExtendType=3 경우 종료기간 날짜 설정 - YYYY-MM-DD 형식으로 입력 |
