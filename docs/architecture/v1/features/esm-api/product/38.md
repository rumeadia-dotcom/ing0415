# 판매자지급 스마일캐시 API

> 출처: https://etapi.gmarket.com/38
> 섹션: 상품 API
> Tistory post id: `38`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| 등록 : [POST] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/customer-benefit/cashback  수정 : [PUT] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/customer-benefit/cashback  해제 : [DELETE] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/customer-benefit/cashback  조회 : [GET] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/customer-benefit/cashback |
| 판매자 부담으로 구매자에게 스마일캐시 혜택을 제공할 수 있습니다. (결제 금액에서 배송비 제외 기준)  구매자에게 부여된 스마일캐시 금액은 판매대금 정산 시 차감됩니다.  0.5%~50%까지 정률로 설정할 수 있으며 상품페이지에 스마일캐시 적립률이 노출됩니다. |

### Request Description

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 항목 | 항목명 | 필수여부 | Type | Description |
|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| goodsNo | 마스터 상품번호 | Y | path |  |
| gmarketRatio | G마켓 적립률 | Y | number | - 적립률 범위 : 0.5%~50% - 0.1% 단위로 입력 가능 |
| auctionRatio | 옥션 적립률 | Y | number | - 적립률 범위 : 0.5%~50% - 0.1% 단위로 입력 가능 |

```
JSON
등록/수정
{
  "gmarketRatio": 0,
  "auctionRatio": 0
}

해제 - URL만 호출/Request Body없음
[DELETE] https://sa2.esmplus.com/item/v1/goods/1139546520/customer-benefit/cashback

조회 - URL만 호출/Request Body없음
[GET] https://sa2.esmplus.com/item/v1/goods/1139546520/customer-benefit/cashback
```

### Response Description

|  |  |  |  |
| --- | --- | --- | --- |
| 항목 | 항목명 | Type | Description |
| goodsNo | 마스터상품번호 | int | 조회시 내려가는 정보 |
| resultCode | 결과코드 | int | 실패 시 내려가는 정보 |
| message | 메시지 | string | 실패 시 내려가는 정보 |
