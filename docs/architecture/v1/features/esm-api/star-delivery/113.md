|  |
| --- |
| Description |
| [POST] https://sa2.esmplus.com/efm/v1/api/service/GetWarehouseOutCapacity |
| 스타배송 물류센터 창고별 반품수용량을 조회하는 API입니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| StorageType | 창고구분 | Y | short | 조회할 창고구분코드 입력 VF창고코드는 담당자별도문의 3: 스타배송 제2물류센터 1층 4: 스타배송 제2물류센터 3층 6: 스타배송 제3물류센터 1층 7: 스타배송 제3물류센터 2층 13: 스타배송 동탄메가센터 3층 3L 14: 스타배송 동탄메가센터 3층 3R 15: 스타배송 동탄메가센터 4층 4L 16: 스타배송 동탄메가센터 4층 4R |
| StartDate | 조회시작일자 |  | Date-Time | 검색조건 일자 조회일 경우 필수 |
| EndDate | 조회종료일자 |  | Date-Time | 검색조건 일자 조회일 경우 필수 |
| pageIndex | 페이지번호 | Y | Int | 정수: Page 구분 |
| pageSize | 페이지사이즈 | Y | Int | 1페이지에 출력하려는 리스트 Max값 1000까지 지원 가능 |

```
JSON
{
  "StorageType": 0,
  "StartDate": "2021-02-19T00:01:51.459Z",
  "EndDate": "2021-02-19T00:01:51.459Z",
  "PageIndex": 1,
  "PageSize": 100
}
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
|  |  |  |  |
| --- | --- | --- | --- |
| ResultCode | 결과코드 | int | 성공: 0 실패 :Error Massage 참고 |
| Message | 메시지 | string |  |
| Data > TotalCount | 전체대상건수 | Int | 조회조건 범위 전체 건수 |
| Data > PageIndex | Response PageNo | Int | 조회시 입력한 PageNo |
| Data > PageSize | Response PageSize | Int | 조회시 입력한 PageSize |
| Data > OutWishDate | 반출희망일자 | DateTime | 오전일 경우 00시, 오후일 경우 12시로 내려감 |
| Data > ParcelCapacity > SellerQty | 수용가능한 셀러 수 | int | 0 : 등록마감일 경우 N : 수용가능한 숫자 |

### **Error Code**

|  |  |  |
| --- | --- | --- |
| **결과코드** | **메시지** | **원인** |
|  |  |  |
| --- | --- | --- |
| 1400 | 창고구분은 필수값입니다. | StorageType > 미입력 혹은 0 입력 시 |
| 1400 | 유효하지 않은 창고거나 판매자의 창고구분이 아닙니다. 등록된 SKU의 창고구분만 조회가 가능합니다. | StorageType > 존재하지 않는 창고번호거나 판매자 SKU 대표창고가 아닌 경우 |
| 1400 | 조회일자를 확인해주세요. 시작일자가 유효하지 않습니다. | StartDate > SearchType이 일자(DATE)일 때 시작일자 미입력 시 |
| 1400 | 조회일자를 확인해주세요. 종료일자가 유효하지 않습니다. | EndDate > SearchType이 일자(DATE)일 때 종료일자 미입력 시 |
| 1400 | 조회일자를 확인해주세요. 종료일자는 시작일자보다 과거일 수 없습니다. | EndDate > 종료일자가 시작일자보다 과거일 경우 |
| 1400 | 검색어가 필수인 검색조건입니다. 검색어를 입력해주세요. | SearchType > 검색어가 필요한 검색조건(입고번호, SKU번호 등)일 때 검색어 미입력 시 |
