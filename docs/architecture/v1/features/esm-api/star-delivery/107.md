| 풀필먼트 스타배송 물류센터로 입고등록한 입고번호를 요청취소하는 API입니다. |

|  |  |  |  |  |
| --- | --- | --- | --- | --- |

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| WarehouseNo | 입고번호 | Y | long | 입고요청취소할 입고번호 |

```

{
  "WarehouseNo": 0
}
```

### **Error Code**

|  |  |  |
| --- | --- | --- |
| **결과코드** | **메시지** | **원인** |
|  |  |  |
| --- | --- | --- |
| 1400 | 이미 취소된 입고번호입니다. | 이미 입고취소한 입고번호를 재처리 요청 시 |
| 1400 | 입고요청취소할 수 없는 상태입니다. | 처리요청한 입고번호가 입고요청취소할 수 없는 상태인 경우 |
| 1400 | 유효하지 않은 입고번호입니다. | 존재하지 않는 입고번호인 경우 |
| 1400 | 판매자의 입고번호가 아닙니다. | 판매자의 입고번호가 아닐 경우 |
| 1400 | 입고예정일자가 지난 입고건은 취소할 수 없습니다. | 입고예정일자가 지난 경우 |
| 1400 | 입고예정일자 기준 D-2일까지만 취소가 가능합니다. | 일반 판매자에 한해, 입고일 D-2 취소 불가 |
