# 외부주문 삭제 API

> 출처: https://etapi.gmarket.com/98
> 섹션: 스타배송 API
> Tistory post id: `98`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| [POST] https://sa2.esmplus.com/efm/v1/api/service/DeleteOuterOrder |
| 등록한 외부주문을 삭제합니다. 출고요청 전, 외부주문을 삭제하면 출고요청 대상에서 제외되며, 외부주문 출고요청 후 취소처리 원하실 경우에는 외부주문 출고취소 API를 호출하여 취소처리합니다. 출고요청된 외부주문은 삭제할 수 없습니다. |

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| SmileOrderNo | 스마일주문번호 |  | long | 삭제할 스마일주문번호 입력 |
