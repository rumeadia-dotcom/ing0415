|  |
| --- |
| Description |
| [POST] https://sa2.esmplus.com/efm/v1/api/service/RemoveSku |
| 등록한 SKU 정보를 삭제합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| SkuId | SKU코드 |  | int | 삭제할 SKU코드 입력 |

```
JSON
{
  "SkuId": 0
}
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
|  |  |  |  |
| --- | --- | --- | --- |
| ResultCode | 결과코드 | int | 성공: 0 실패 :Error Massage 참고 |
| Message | 메시지 | string |  |
| Data | Data | string |  |
| Data > SkuId | SKU코드 | string | 처리된 SKU코드 Return |

 JSON\_Success

```
{
  "ResultCode": 0,
  "Message": "",
  "Data": {
    "SkuId": 0
  }
}
```

#### JSON\_Fail

```
{
    "ResultCode": int,
    "Message": "string",
    "Data": null
}
```

#### Sample Code\_Success

```
{
    "ResultCode": 0,
    "Message": "success",
    "Data": {
        "SkuId": 42032
    }
}
```

#### Sample Code\_Fail

```
{
    "ResultCode": 1400,
    "Message": "검수대기/검수거부/입고대기 상태의 SKU만 삭제 가능합니다. 현재상태 : SD100(삭제)",
    "Data": null
}
```

|  |  |  |
| --- | --- | --- |
| **결과코드** | **메시지** | **원인** |
|  |  |  |
| --- | --- | --- |
| 1400 | 검수대기/검수거부/입고대기 상태의 SKU만 삭제 가능합니다. 현재상태 : 현재상태코드(현재상태명) | SkuId > 처리요청한 SKUId가 삭제 가능한 상태가 아닐 경우 |
