| Description |
| 상품등록 : [POST] https://sa2.esmplus.com/item/v1/goods 상품수정 : [PUT] https://sa2.esmplus.com/item/v1/goods/{goodsNo} 상품조회 : [GET] https://sa2.esmplus.com/item/v1/goods/{goodsNo} |
| G마켓/옥션 2.0 풀필먼트 스타배송 상품을 등록합니다. 상품등록 성공시 바로 판매 가능한 상태로 진행되고, 실패시 실패 사유에 따라 정보 확인 및 수정 통해 재등록 필요합니다.  상품이 등록되면 일괄 수정으로 수정 진행할 수 있고 기능별 수정 API를 사용하여 부분 정보 수정이 가능합니다. 수정 API는 전체 정보 일괄 수정할 경우에만 사용하도록 하고 그 외는 기능별 수정 가능한 Partial API로 각 정보를 수정합니다. |
