# 배송비 정산조회 API

> 출처: https://etapi.gmarket.com/42
> 섹션: 정산조회 API
> Tistory post id: `42`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| [POST] https://sa2.esmplus.com/account/v1/settle/getsettledeliveryfee |
| 배송비 정산데이터를 조회합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
| SiteType | 사이트구분 | Y | string | 처리하려는 주문번호 사이트구분 선택  A : 옥션 G : G마켓 |
| PackNo | 장바구니번호 |  | long | SiteType > "G마켓"일 경우는 장바구니번호 입력 후 조회 |
| DeliveryGroupNo | 묶음배송번호 |  | bigint | SiteType > "옥션"일 경우는 묶음배송번호, 장바구니번호 모두 입력 후 조회 |
| TransferNumber | 송금번호 |  | int | 글로벌셀러 예치금 송금번호 입력 |
| SrchType | 기간검색 구분 | Y | string | D1 : 입금확인일 -정상 D3 : 매출마감일(일반적으로는 배송완료일 익일, 환불건일 경우 차이발생 가능) D6 : 송금일(당일데이터는 영업일 기준 D+1일로 조회가능함) D7 : 환불일 -환불 D8 : 입금확인일(옥션은 입금확인되었으면서 실제 송금이 발생된 주문건 기준)+환불일 D10 : 글로벌셀러 예치금 송금일 |
| SrchStartDate | 조회기준 시작일 | Y | date | YYYY-MM-DD형태 |
| SrchEndDate | 조회기준 종료일 | Y | date | YYYY-MM-DD형태 |
| PageNo | 페이지번호 | Y | int |  |
| PageRowCnt | 1Page Max | Y | int |  |

}

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
| ResultCode | 결과 | int | 성공 : Success 실패 : Error Massage 참고 |
| Message | 메시지 | stirng |  |
| SiteType | 사이트구분 | string | A : 옥션 G : G마켓 |
| TotalCount | 총건수 | int | 조회된 총 건수 |
| TotalDelFeeAmt | 배송비 금액 합계 | Number | 전체 조회 내역 내 DelFeeAmt 합계 |
| SellerId | 판매자ID | string |  |
| PageSize | 페이지 사이즈 | int |  |
| PageNo | 페이지 번호 | int |  |
| Data > PackNo | 장바구니 번호 | int | 정산데이터에서 장바구니 번호는 PackNo |
| Data > PayDate | 입금 확인일 | Date | 옥션 주문 경우 DelFeeType > 30이면 데이터 내려가지 않고 Default 값 내려감 (0001-01-01T00:00:00)  G마켓 주문일 경우 DelFeeType > 70이면 데이터 내려가지 않고 Default 값 내려감 (0001-01-01T00:00:00) |
| Data > RefundDate | 배송비 환불일 | Date |  |
| Data > RemitDate | 배송비 송금일 | Date |  |
| Data > RevenueDate | 매출 마감일 | Date |  |
| Data > DelFeeAmt | 배송비 금액 | Number |  |
| Data > DelFeeCommission | 배송비 수수료 금액 | Number |  |
| Data > DelFeeNo | 배송비 결제번호 | int |  |
| Data > DelFeePayWay | 배송비 지급수단 | string | A : 계좌송금 B : 판매자예치금 S : 스마일캐시 |
| Data > Kind | 주문 구분 | string | 1 : 원주문 2 : 환불 |
| Data > DeliveryGroupNo | 배송비 그룹번호 | bigint |  |
| Data > DelFeeType | 배송비 구분 | string | 10 : 원배송비 20 : 추가배송비 30 : (G)사용한원배송비환불금차감 (A)원배송비(최초배송비)외 추가배송비 등 환불금 금액과 거래완료 후 환불에 따른 배송비 집계 40 : 반품배송비 50 : 추가반품배송비 60 : 무료반품배송비 70 : 교환배송비 |
| Data > RefContrNo | 대표 주문번호 | int | - (G마켓만 제공) 해당 배송비의 대표주문번호 값 내려감 - 배송비 유형에 따라 대표 주문번호가 있는 경우 내려가고 송금일/환불일 기준으로만 조회됨 |
| Data > TransferNumber | 글로벌셀러 예치금 송금번호 | int |  |
| Data > TransferDate | 글로벌셀러 예치금 송금일 | Date |  |
| Data > ExchangeRate | 글로벌셀러 예치금 송금시 환율 | Number |  |
