| 배송Slot 기준으로 원거리 지원하는 Slot 인지 여부를 등록합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
| branchId | 지점코드 | Y | path |  |
| deliverySlotId | 배송slot코드 | Y | path |  |
| zipcode | 우편번호 | Y |  |  |

```
JSON
{
  "zipcode": "string"
}
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
| Zipcode | 우편번호 | int | 조회 시 내려가는 정보 |
| deliverySlotId | 원거리여부 | int | 조회 시 내려가는 정보 |
| resultCode | 결과코드 | int | 실패 시 내려가는 정보 |
| message | 메시지 | string | 실패 시 내려가는 정보 |

 JSON\_Success

```
{
  "zipcode": "string",
  "deliverySlotId": 0
}
```

#### JSON\_Fail

```
{
   "resultCode": 0,
   "message": "string"
}
```

#### Sample Code\_Success

```
등록/수정
{
  "zipcode": "12345"
}

조회
[
    {
        "zipcode": "06131",
        "deliverySlotId": 348824
    }
]
```

#### Sample Code\_Fail

```
{
  "resultCode":1000,
  "message":"해당 우편번호는 존재하지 않습니다(1)"
}
```

### **Error Code**

|  |  |  |
| --- | --- | --- |
| **결과코드** | **메시지** | **원인** |
|  |  |  |
| --- | --- | --- |
| 1000 | 해당 배송 슬롯은 다른 지점의 배송 슬롯입니다 | 다른 지점의 배송슬롯을 등록 시 발생 |
| 1000 | 해당 우편번호는 존재하지 않습니다 | 등록하지 않았던 우편번호 사용 시 발생 |
| 1000 | 해당 배송Slot은 존재하지 않습니다 | 원거리 지원하지 않는 배송슬롯에 등록시도 시 발생 |
| 1000 | 해당 배송Slot은 존재하지 않습니다 | 잘못된 배송 slot 삭제 시도시 발생 |
| 1000 | 당일 배송관 브랜치 우편번호 템플릿 정보가 없습니다. | 배송정보가 없을 때 발생 |
| 1000 | 해당 우편번호는 존재하지 않습니다(1) | 잘못된 우편번호 등록 시 발생 |
| 1000 | 유효하지 않은 지점 입니다.(해당 지점은 존재하지 않습니다) | 잘못된 지점번호 등록 시 발생 |
