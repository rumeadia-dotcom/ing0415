# 상품별 프로모션 관리 API

> 섹션: 서비스 API
> Tistory post id: `81`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| 등록/수정 : [PUT] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/exshop  조회 : [GET] https://sa2.esmplus.com/item/v1/goods/{goodsNo}/exshop |
| 상품별 프로모션 코드 및 중량 정보를 연동합니다. 프로모션 코드 연동시 프론트에서 프로모션 아이콘이 노출됩니다. 프로모션 코드는 별도로 제공합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| goodsNo | 마스터 상품번호 | Y | path |  |
| promotionCode1 | 행사코드1 |  | int |  |
| promotionCode2 | 행사코드2 |  | int |  |
| totalWeight | 총중량 |  | number |  |
| unitPrice | 중량당가격 |  | number |  |
| weightUnit | 중량단위 |  | string | Ex) g, kg, EA, L, ml, cc |

```
JSON
{
  "promotionCode1": int,
  "promotionCode2": int,
  "totalWeight": 0,
  "unitPrice": 0,
  "weightUnit": "string"
}
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
| goodsNo | 마스터 상품번호 | int | 조회 시 내려가는 정보 |
| resultCode | 결과코드 | int | 실패 시 내려가는 정보 |
| message | 메시지 | string | 실패 시 내려가는 정보 |

 JSON\_Success

```
{
  "goodsNo": 0
}
```

#### JSON\_Fail

```
{
   "resultCode": 0,
   "message": "string"
}
```

#### Sample Code\_Success

```
{
    "goodsNo": 1139546520
}
```

#### Sample Code\_Fail

```
{
    "resultCode": 1000,
    "message": "등록된 상품이 존재 합니다."
}
```
