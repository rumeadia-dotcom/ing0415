# 실시간 가격, 재고 체크 - G마켓용

> 섹션: 서비스 API
> Tistory post id: `86`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| **URL은 제휴사에서 개발 후 전달 필요** |
| 상품상세페이지에서 고객이 장바구니 or 구매하기 버튼 클릭 실시간 주문체크 API 호출하여 실제 주문 가능한 상태인지 체크합니다 실시간 재고 체크를 위하여 주문수량, 선택옵션정보를 제휴사 재고체크 URL로 XML형식으로 전송해 드립니다.   제휴사에서는 XML을 Request 하여 상품 가격/재고 체크를 하고 주문가능여부를 XML형식으로 응답 주셔야 합니다. 응답주신 결과로 고객에게 안내 합니다. (주문가능할 경우 별도 Notice없이 주문진행시킴, 무응답이어도 주문진행됨)   반드시 아래 규약과 동일하게 구현되어야 하며 본 API는 양사 개발 필요하므로 제한적으로 제공합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
| ORDER\_INFO > PRODUCT NO | 판매자관리번호 | Y | string |  |
| ORDER\_INFO > ORDER\_AMT | 주문수량 | Y | int |  |
| ORDER\_INFO > PRICE | 상품금액(판매가) | Y | int |  |
| ORDER\_INFO > GD\_NO | G마켓 Site상품번호 | Y | int |  |
| ORDER\_INFO > BRANCHID | 당배관 지점코드 |  | string | 당일배송(장보기) 상품만 해당된 일반 상품은 해당 없음 |
| ORDER\_INFO > ORDER\_OPTION > OPTION\_INFO > NAME | 옵션명 | Y | string | 옵션정보가 있을 경우 |
| ORDER\_INFO > ORDER\_OPTION > OPTION\_INFO > VALUE | 옵션값 | Y | string | 옵션정보가 있을 경우 |
| ORDER\_INFO > ORDER\_OPTION > OPTION\_INFO > PRICE | 옵션금액 | Y | int | 옵션정보가 있을 경우 |
| ORDER\_INFO > ORDER\_OPTION > OPTION\_INFO > AMT | 옵션수량 | Y | int | 옵션정보가 있을 경우 |
| ORDER\_INFO > ORDER\_OPTION > OPTION\_INFO > STOCK\_NO | 판매자 옵션관리 번호 |  | string |  |

```
옵션정보 있을 경우
<?xml version="1.0" encoding="utf-8" ?>
<ORDER_INFO>
      <PRODUCT NO="123456789" ORDER_AMT="2" PRICE="10000" GD_NO="21234567890" />
      <ORDER_OPTION>
          <OPTION_INFO>
                <NAME><![CDATA[사이즈]]></NAME>
                <VALUE><![CDATA[55]]></VALUE>
                <PRICE>1000</PRICE>
                <AMT>10</AMT>
                <STOCK_NO>00001</STOCK_NO>
          </OPTION_INFO>
      </ORDER_OPTION>
</ORDER_INFO>

옵션정보 없을 경우
<?xml version="1.0" encoding="utf-8" ?>
<ORDER_INFO>
      <PRODUCT NO="123456789" ORDER_AMT="2" PRICE="10000" GD_NO="21234567890"  />
</ORDER_INFO>
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
| STOCK\_REMAIN\_INFO > PRODUCT NO | 판매자관리번호 | int |  |
| ORDER\_INFO > REMAIN\_YN | 주문가능여부 | string | Y : 주문가능 N : 주문불가 주문불가로 Return할 경우 주문불가 관련 알럿 띄우고 주문 진행시키지 않음 응답이 지연될 경우(약5초) 무응답으로 간주하여 주문진행 |
| STOCK\_REMAIN\_INFO > ORDER\_OPTION > OPTION\_INFO > NAME | 옵션명 | string | 옵션정보가 있을 경우 |
| STOCK\_REMAIN\_INFO > ORDER\_OPTION > OPTION\_INFO >VALUE | 옵션값 | string | 옵션정보가 있을 경우 |
| STOCK\_REMAIN\_INFO > ORDER\_OPTION > OPTION\_INFO > REMAIN\_YN | 옵션별 주문가능여부 | string | 옵션정보가 있을 경우 |
| STOCK\_REMAIN\_INFO > ORDER\_OPTION > OPTION\_INFO > REMAIN\_YN | 옵션 재고 수량 | int |  |
| STOCK\_REMAIN\_INFO > ORDER\_OPTION > OPTION\_INFO > STOCK\_NO | 판매자 옵션관리 번호 | string |  |

```
<STOCK_REMAIN_INFO>
    <PRODUCT NO="123456789" REMAIN_YN="Y" />
</STOCK_REMAIN_INFO>
```
