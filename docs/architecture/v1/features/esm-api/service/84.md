# 실시간 Slot체크 API

> 출처: https://etapi.gmarket.com/84
> 섹션: 서비스 API
> Tistory post id: `84`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| **URL은 제휴사에서 개발 후 전달 필요** |
| 주문시 선택한 배송Slot 배송 가능한지 여부를 체크합니다. 주문 가능 응답시, 무응답시, 응답 지연시 주문진행 되며 주문 불가 응답시에만 주문 불가하게 됩니다. 아래 규약과 동일하게 구현되어야 하며 본 API는 양사 개발 필요하므로 제한적으로 제공됩니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| SellCustNo | 판매자번호 | Y | int | 관리자에게 확인 필요 |
| BranchCode | 지점코드 | Y | int | 고객이 선택한 지점의 코드가 내려감 |
| RequestDate | 배송날짜 | Y | date-time | 고객이 선택한 배송날짜가 내려감 |
| SlotNo | 배송Slot코드 | Y | int | 고객이 선택한 배송Slot 코드가 내려감 |

```
JSON
Method : Post
[
    {
       "SellCustNo":"117205821",
       "BranchCode":"100000025",
       "RequestDate":"2018-03-23",
       "SlotNo":110001300
    }
]
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
|  |  |  |  |
| --- | --- | --- | --- |
| SellCustNo | 판매자번호 | int |  |
| BranchCode | 지점코드 | int |  |
| RequestDate | 배송날짜 | date-time |  |
| SlotNo | Slot코드 | int |  |
| DeliveryEnableYn | 배송가능여부 | String | Y : 배송가능 N : 배송불가 배송불가 시 주문 진행시키지 않음 무응답이거나 읍답 지연 시, 주문진행함 |
