| 홈쇼핑 실시간 라이브 편성표를 구성하는 API입니다.  실시간 방송상품 및 대체상품 등 라이브 편성표에 구성할 정보를 관리합니다. 방송일/방송시간대/방송시작시간에 대해 유효성 체크하며, 아래 항목에 대해서 다른 방송정보가 등록 될 경우, 최신 업데이트 정보 기준으로 서비스 제공합니다. (ProgramID, programType & ~~broadcastProgramType & broadcastProgramDetailType~~) 등록 시, 일자별 데이터를 확인하여 제휴사에서 관리 필요합니다. 편성표 삭제 시 동일 ProgramID의 모든정보가 일괄 삭제됩니다. |
