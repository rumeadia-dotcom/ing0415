# 미수령신고조회 API

> 출처: https://etapi.gmarket.com/74
> 섹션: 클레임 API
> Tistory post id: `74`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| **[POST] https://sa2.esmplus.com/shipping/v1/Delivery/ClaimList** |
| 미수령신고 주문을 조회합니다. 미수령신고된 주문은 정산되지 않기 때문에 구매자와 연락하여 배송사고가 발생했는지 등 확인해야 합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| SearchType | 조회기준구분 | Y | string | 0: 주문번호 조회 1: 미수령신고일 조회 |
| orderNo | 주문번호 | Y | long | 조회하고자 하는 주문번호 입력 |
| StartDate | 조회기준시작일 | Y | date | YYYY-MM-DD형태 |
| EndDate | 조회기준종료일 | Y | date | YYYY-MM-DD형태 |

```
JSON
{
  "SearchType":0,
  "OrderNo":0,
  "StartDate":"2019-04-11",
  "EndDate":"2019-04-11"
}
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
| ResultCode | 결과 | string | 성공: Success 실패: Error Massage 참고 |
| Message | 메시지 | string |  |
| TotalCount | 총조회건수 | int |  |
| Data > OrderNo | 주문번호 | long | 처리된 주문번호 Return |
| Data > ClaimDate | 미수령신고일자 | String |  |
| Data > ClaimReason | 미수령신고사유 | date | \*고정 Text로 값 내려감 [G마켓] 배송지연7일이상 [옥션] 일부상품 누락 상품전체 미도착 |
| Data > ClaimSolveDate | 미수령철회요청일 | date |  |

### **Error Code**

|  |  |  |
| --- | --- | --- |
| **결과코드** | **메시지** | **원인** |
|  |  |  |
| --- | --- | --- |
| 3000 | 조회 기간은 30일 이내여야 합니다 | requestDateFrom~requestDateto 조회기간 초과 |
| 1100 | 데이터가 없습니다 | 조회조건에 맞는 미수령신고 주문 건 없을 경우 |
