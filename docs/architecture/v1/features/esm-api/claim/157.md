| Description |
| [POST] https://sa2.esmplus.com/claim/v1/sa/Cancel/{orderNo}/AfterRemittanceBySeller |
| 옥션 주문 중 거래완료(정산 완료) 후 환불 필요한 경우 처리하는 API 입니다. (지마켓은 불가) 자동 수거지시와 배송비 청구가 불가하기 때문에 처리 전 상품 수거는 별도 확인 해야 하며 상품 발송/회수로 인해 발생 배송비는 구매자와 협의 후 진행 되어야 합니다. 처리 이후에는 고객에게 환불 알림이 발송됩니다. 환불할 금액이 예치금 내 있어야 처리 가능하며 예치금 부족시 충전 후 처리 하시길 바랍니다. |
