|  |
| --- |
| Description |
| **[PUT] [https://sa2.esmplus.com/claim/v1/sa/exchange/](https://sa.esmplus.com/claim/v1/sa/exchange/){orderNo}/pickup** |
| 교환수거송장의 배송완료처리합니다. 교환수거송장의 배송완료가 정상적으로 처리되지 않으면 정산 누락될 수 있습니다. 택배 Tracking 되지 않는 주문의 경우 해당 수거완료처리를 진행합니다. 옥션의 경우 재발송 송장처리를 하면 자동으로 교환수거 완료처리되므로, 교환수거 송장처리 ->교환수거 완료처리 -> 재발송 송장처리 -> 재발송 배송완료처리 순으로 처리합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| OrderNo | 주문번호 | Y | Path | 처리하려는 주문번호 입력 |
| PickupCompleteDate | 수거완료일자 |  | string | 배송완료된 일자 YYYY-MM-DD 형식 |
| SiteType | 사이트구분 | Y | int | 처리하려는 주문번호 사이트구분 선택  1 : 옥션 2 : G마켓 |

```
JSON
{
  "SiteType": int,
  "PickupCompleteDate": "2019-04-22"
}
```
