# 판매자 직접 반품 신청 API

> 출처: https://etapi.gmarket.com/54
> 섹션: 클레임 API
> Tistory post id: `54`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| Description |
| **[POST] https://sa2.esmplus.com/claim/v1/sa/Return/{orderNo}/Request** |
| 판매자가 직접 반품 처리시 호출하는 API 입니다. 반품 요청 상태로 주문 상태 변경 되며 반품 완료 처리시에는 반품 승인 API 호출 되어야 합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
| orderNo | 주문번호 | Y | path |  |
| siteType | 사이트 구분 | Y | int | 1 : 옥션 2 : 지마켓 |
| payNo | 장바구니(결제) 번호 | Y | long |  |
| reason | 반품 사유 | Y | string | NC : 구매자변심 BG : 선택사항 변경 RG : 상품 파손 OG : 다른 상품 오배송 DD : 상품 미도착 |
| itemStatus | 상품 상태 | Y | string | OP : 개봉후 미사용 NP : 미개봉 UP : 개봉 후 사용 |
| alreadySent | 구매자 이미 보냄 여부 | Y | boolean | true : 고객이미 보냄 false : 수거요청 \*발송 택배사가 CJ, 한진, 우체국, 롯데, 로젠 경우만 수거요청 가능 |
| pickupCompCode | 수거 택배사 코드 |  | int |  |
| invoiceNo | 수거 송장 번호 |  | string |  |

```
JSON
{
  "siteType": int,
  "payNo": "long",
  "reason": "string",
  "itemStatus": "string",
  "alreadySent": boolean,
  "pickupCompCode": int,
  "invoiceNo": "string"
}
```

| Message | 메시지 |  | string |  |
