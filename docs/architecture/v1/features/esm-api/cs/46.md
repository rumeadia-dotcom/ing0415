# 긴급알리미 답변 API

> 출처: https://etapi.gmarket.com/46
> 섹션: CS API
> Tistory post id: `46`
> 추출 시점: 2026-05-29

---

|  |
| --- |
| **Description** |
| [POST] https://sa2.esmplus.com/assist/v1/Selling/AddEmergencyInformReply |
| 지마켓, 옥션 CS에서 문의한 내용을 CS 또는 상품 담당자가 답변합니다. |

### **Request Description**

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **항목** | **항목명** | **필수여부** | **Type** | **Description** |
| EmerMessageNo | 문의번호 | Y | string | 조회에서 발급받은 문의번호 연동 |
| answerStatus | 답변상태 | Y | int | 보내는 답변의 처리상태 연동  1 : 처리중 2 : 처리완료 |
| Comments | 답변내용 | Y | string | 답변 내용 연동 |

```
JSON
{
  "emerMessageNo": "string",
  "comments": "string",
  "answerStatus": int
}
```

### **Response Description**

|  |  |  |  |
| --- | --- | --- | --- |
| **항목** | **항목명** | **Type** | **Description** |
|  |  |  |  |
| --- | --- | --- | --- |
| ResultCode | 결과코드 | int | 실패 시 내려가는 정보 |
| message | 메시지 | string | 실패 시 내려가는 정보 |
| data | 데이터 | object | 실패 시 내려가는 정보 |
| Data > EmerMessageNo | 문의번호 | string | 조회 시 내려가는 정보  답변 성공한 문의번호 Return(옥션만) |

 JSON\_Success

```
{
   "ResultCode":0,
   "Message":"string",
   "Data":
   {
     "EmerMessageNo": "string"
 }
}
```

#### JSON\_Fail

```
{
    "ResultCode": int,
    "Message": "string",
    "Data": null
}
```

#### Sample Code\_Success

```
{
   "ResultCode":0,
   "Message":"성공",
   "Data":
   {
      "EmerMessageNo":3578427
   }
}
```

#### Sample Code\_Fail

```
{
    "ResultCode": 2141,
    "Message": "입력된 문의번호에 해당하는 정보가 존재하지 않습니다.",
    "Data": null
}
```

### **Error Code**

|  |  |  |
| --- | --- | --- |
| **결과코드** | **메시지** | **원인** |
|  |  |  |
| --- | --- | --- |
| 2141 | Request 형식에 문제가 있습니다. | emerMessageNo 오입력 or 미입력 했을 경우 |
| 20 | 답변상태 입력값은 1(처리중), 2(처리완료) 중 하나로 입력가능합니다. | answerStatus 지원하지 않는 값 연동했을 경우 |
| 100 | 이미 처리 완료된 건입니다. | 처리완료된 메시지 답변 재호출할 경우 |
| 20 | 입력 답변 내용이 없습니다. 답변 내용을 확인하여 주세요. | Comments 미입력 했을 경우 |
| 20 | 번호는 0보다 커야합니다. | emerMessageNo : 0 으로 호출했을 경우 |
| 2141 | 입력된 문의번호에 해당하는 정보가 존재하지 않습니다. | 판매자의 문의번호가 아닐경우 |
