## 개요

구매자를 대신하여 반품신청을 합니다. 이메일, 전화통화 등으로 구매자의 동의를 얻은 후 진행해주세요.
구매자의 동의 없이 판매불가처리를 하여 발생하는 구매자 불만사항 에 대해서는 11번가가 책임지지 않습니다.
