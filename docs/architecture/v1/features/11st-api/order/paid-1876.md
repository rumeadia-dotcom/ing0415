## 개요

조회 기간은 최대 7일(1주일)입니다.
결제완료되어 발송을 기다리는 주문상품입니다.
상품재고, 묶음배송상품, 추가구성상품 등을 확인하시고 발송가능한 상품이면 [발주확인] 처리하십시오.
[발주확인]된 주문에 대해서는 구매자가 임의로 주문을 취소하거나, 배송지정보를 변경할 수 없습니다. 단, [발주확인] 이후 주문취소는 구매자가 주문 취소요청을 하고 판매자의 취소승인을 통해 진행됩니다. ‘취소/반품/교환 > 취소관리’에서 취소요청 주문을 확인해주십시오.
품절 등 판매자 귀책사유로 인한 [판매불가] 처리시에는 ‘판매거부’로 간주되어 판매자의 신용점수가 1점 차감됩니다.
최대 3,000건까지 조회 가능합니다.
