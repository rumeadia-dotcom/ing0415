# 수량 분리발송처리 (1개의 주문상품 여러개 나눠서 송장등록 가능)

## 개요

발송처리할 내역에서 읽어온 주문 중, 1개의 주문상품의 여러개의 송장으로 발송처리 해야할경우 해당API를 이용해주세요.

배송주체가 "업체배송"인 주문만 해당 API를 이용하셔야 합니다.
배송방법 및 송장번호를 입력하시고 [발송완료] 처리를 하시면 발송내용이 구매자에게 안내 됩니다.
택배/퀵/우편/등기/직접배송 등 상품의 배송수단과 운송장정보를 입력하셔야 합니다.
품절 등 판매자 귀책사유로 인한 [판매불가] 처리시에는 ‘판매거부’로 간주되어 판매자의 신용점수가 1점 차감됩니다.
