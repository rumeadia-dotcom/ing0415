# [즉시할인쿠폰] 요청상태 확인

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360033685834>
> Zendesk article id: `360033685834`
> 섹션: 쿠폰 / 캐시백 APIs
> 추출 시점: 2026-05-28

---

**API 적용 가능한 구매자 사용자 지역: 한국**

requestedId를 통해 즉시할인 쿠폰 API의 요청 결과를 확인합니다.  
쿠폰생성, 쿠폰파기, 쿠폰아이템 생성, 쿠폰아이템 파기 API 요청 시에  
반환되는 requestedId 값을 가지고 성공/실패(DONE/FAIL) 결과를 확인할 때 사용합니다.

## Path

GET

/v2/providers/fms/apis/api/v1/vendors/{vendorId}/requested/{requestedId}

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/fms/apis/api/v1/vendors/A00000001/requested/649102321051192483

## Request Parameters

### Path Segment Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| vendorId | | | | | O | String | 판매자ID  쿠팡에서 업체에게 발급한 고유 코드  예) A00012345 |
| requestedId | | | | | O | Number | 결과를 조회할 ID |

### Request Example

not require body

### Error Spec
