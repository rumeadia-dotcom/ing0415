# [다운로드쿠폰] 생성

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360034205493>
> Zendesk article id: `360034205493`
> 섹션: 쿠폰 / 캐시백 APIs
> 추출 시점: 2026-05-28

---

**API 적용 가능한 구매자 사용자 지역: 한국**

고객이 다운받아 사용할 수 있는 쿠폰을 생성합니다.  
  
쿠폰이 붙은 상품의 총 주문액이 판매자가 설정한 일정금액을 충족해야 사용가능합니다.  
쿠폰은 정률/정액할인으로 발급할 수 있으며, 동일 쿠폰에 최대 3개의 정책을 생성할 수 있습니다.  
(예: 10,000원 구매시 1천원 할인, 20,000원 구매시 3천원 할인, 30,000원 구매시 3천원 할인 등)  
  
한 개의 옵션ID(vendoritem)당 하나의 쿠폰을 생성할 수 있으며, 생성 후 최소 1시간 이후부터 프론트에 반영됩니다.  
(만약 오후 3시에 쿠폰을 생성한다면, 쿠폰 유효기간으로 설정할 수 있는 가장 빠른 시간은 같은 날 오후 4시입니다.)

다운로드쿠폰은 최초 생성 시 설정한 쿠폰 적용 상품을 추후 바꿀 수 없습니다.  
만약 쿠폰을 적용할 상품을 추가 혹은 삭제하고 싶은 경우에는 기존에 발행한 쿠폰을 중지하고 새로운 쿠폰을 생성해야 합니다.  
최초 쿠폰 생성 시 쿠폰을 적용할 상품을 신중하게 설정해주세요!

## Path

POST

/v2/providers/marketplace\_openapi/apis/api/v1/coupons

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/marketplace\_openapi/apis/api/v1/coupons
