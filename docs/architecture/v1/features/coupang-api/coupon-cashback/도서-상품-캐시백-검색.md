# [도서] 상품 캐시백 검색

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360033645254>
> Zendesk article id: `360033645254`
> 섹션: 쿠폰 / 캐시백 APIs
> 추출 시점: 2026-05-28

---

**API 적용 가능한 구매자 사용자 지역: 한국**

해당 옵션 ID 에 적용된 캐시백 룰 정보를 조회합니다.

캐시백이 적용되지 않은 옵션 ID(vendorItemId)를 조회 시 404 또는 400 에러가 발생합니다.

## Path

GET

/v2/providers/openapi/apis/api/v4/vendors/{vendorId}/products/items/cashback

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/openapi/apis/api/v4/vendors/A00012345/products/items/cashback?ruleId=1&vendorItemId=3000000005

## Request Parameters

### Path Segment Parameter

| vendorId | | | | | O | String | 판매자 ID  쿠팡에서 업체에게 발급한 고유 코드  예) A00012345 |

### Query String Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |

| vendorItemId | | | | | O | Number | 옵션 ID |

### Request Example

not require body

### Error Spec

| HTTP 상태 코드(오류 유형) | 오류 메시지 | 해결 방법 |
| --- | --- | --- |
| 404 (요청변수확인) | Can not find promotion with given condition vendorItemId : 7015\*\*\*6995 | 조회하려는 옵션 ID (vendorItemId) 에 캐시백이 적용 되었는지 확인합니다. |
| 400 (요청변수확인) | VendorItemId can't be null | 옵션 ID (vendorItemId) 를 입력했는지 확인합니다. |
| 400 (요청변수확인) | Invalid vendorItemId, vendorItemId doesn't exist or doesn't belong to corresponding vendorId | 조회하려는 옵션 ID (vendorItemId) 가 올바른지 또는 캐시백이 적용되었는지 확인합니다. |
