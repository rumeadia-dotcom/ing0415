# [다운로드쿠폰] 요청상태 확인

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360034209973>
> Zendesk article id: `360034209973`
> 섹션: 쿠폰 / 캐시백 APIs
> 추출 시점: 2026-05-28

---

**API 적용 가능한 구매자 사용자 지역: 한국**

requestTransactionId 이용하여 쿠폰아이템 생성 API 요청 결과를 확인합니다.

## Path

GET

/v2/providers/marketplace\_openapi/apis/api/v1/coupons/transactionStatus

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/marketplace\_openapi/apis/api/v1/coupons/transactionStatus?requestTransactionId=et5\_154210571558673553106

## Request Parameters

### Query String Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| requestTransactionId | | | | | O | Number | 결과를 조회할 ID |

### Request Example

not require body

## Response Message

| Name | | | | | Type | Description |
| --- | --- | --- | --- | --- | --- | --- |
| transactionStatusResponse | | | | | Object | 호출응답 상세 |
|  | type | | | | String | 상태 확인 대상 API |
|  | total | | | | Number | 요청 vendoritem 수 |
|  | succeeded | | | | Number | 호출 성공 vendoritem 수 |
|  | status | | | | String | 호출 결과 |
|  | requestedId | | | | String | 응답 조회 Key |
|  | couponFailedVendorItemIdResponses | | | | Array | 등록실패 vendoritemId 상세 사유 |
|  |  | vendorItemId | | | Number | API 호출에 사용한 옵션ID |
|  |  | failureReason | | | String | 호출 실패 사유 |
|  | failed | | | | Number | 호출 실패 vendoritem 수 |
|  | couponId | | | | Number | 해당 쿠폰ID |

### Error Spec

| HTTP 상태 코드 (오류 유형) | 오류 메시지 | 해결 방법 |
| --- | --- | --- |
| 500 (요청변수확인) | 시스템 오류 | requestTransactionId 값을 올바로 입력했는지 확인합니다. |

### URL API Name

GET\_REQUEST\_STATUS\_BY\_TRANSACTION\_ID
