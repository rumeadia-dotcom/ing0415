# 이미출고 처리

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360033793034>
> Zendesk article id: `360033793034`
> 섹션: 배송 / 환불 APIs
> 추출 시점: 2026-05-28

---

**API** **적용** **가능한** **구매자** **사용자** **지역****:** **한국**

고객이 주문을 취소처리 하여

반품요청목록 조회 시 취소진행 상태가 출고중지요청(**RELEASE\_STOP\_UNCHECKED**) 이거나 반품접수미확인(**RETURNS\_UNCHECKED**) 이면서 releaseStatus가 "N" 일 경우에 사용가능합니다.   
이미 판매자가 상품을 발송한 상태일 때 사용합니다.

<주의!>고객의 출고중지요청에도 불구하고 상품을 출고했다면 왕복 반품 배송비는 판매자의 귀책입니다.

## Path

PATCH PUT

/v2/providers/openapi/apis/api/v4/vendors/{vendorId}/returnRequests/{receiptId}/completedShipment

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/openapi/apis/api/v4/vendors/{vendorId}/returnRequests/{receiptId}/completedShipment

### Error Spec

| HTTP 상태 코드 (오류 유형) | 오류 메시지 | 해결 방법 |

| 400 (요청변수확인) | 이미 저장된 송장번호가 있어 송장번호 등록이 불가능합니다. (동일송장번호 업로드 조건 : 수취인정보 동일) [합포장번호 : 10275\*\*\*\*15] | 등록한 송장번호가 이미 쿠팡에 등록된 번호인지 확인합니다. [FAQ 참조](https://developers.coupangcorp.com/hc/ko/articles/360022937074) |
