# 장기미배송 배송완료 처리

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360034320713>
> Zendesk article id: `360034320713`
> 섹션: 배송 / 환불 APIs
> 추출 시점: 2026-05-28

---

**API** **적용** **가능한** **구매자** **사용자** **지역****:** **한국**

송장 등록 후 1달이 경과하였으나, 배송추적 결과를 확인할 수 없는 내용에 대해 배송완료로 상태를 변경합니다.

정산된 주문에 대한 미배송으로 인해 고객 분쟁 발생 시 분쟁처리에 대한 비용은 판매자가 부담해야 하므로 정확한 확인이 필요합니다.

배송지시(DEPARTURE) 이후 30일이 지난 배송지시, 배송중 상태인 주문만 처리 가능합니다.

## Path

POST

/v2/providers/openapi/apis/api/v4/vendors/{vendorId}/completeLongTermUndelivery

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/openapi/apis/api/v4/vendors/A00012345/completeLongTermUndelivery

### Error Spec

| HTTP 상태 코드(오류 유형) | 오류 메시지 | 해결 방법 |
| --- | --- | --- |
| 400 (요청변수확인) | 배송완료 처리 실패. 배송상태: INSTRUCT | 상품준비중(INSTRUCT) 상태의 주문은 배송완료 처리를 할 수 없습니다. 배송지시 이후 상태의 주문만 처리 가능합니다. |
| 400 (요청변수확인) | 배송완료 처리 실패. 배송상태: DEPARTURE | 올바른 송장번호인지 확인합니다. 또한 배송지시 이후 30일이 지난 주문 건인지 확인합니다. 배송지시 시각에 따라 32일이 지난 후에 처리 가능할 수 있습니다. |
| 400 (요청변수확인) | 배송완료 처리 실패. 배송상태: NONE\_TRACKING | 업체직송(NONE\_TRACKING) 상태의 주문은 배송완료 처리를 할 수 없습니다. 더 이상 완료처리 시도를 하지 않도록 처리합니다. |
