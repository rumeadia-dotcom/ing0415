**API** **적용** **가능한** **구매자** **사용자** **지역****:** **한국**

매출인식월을 기준으로 지급 확정/예정된 내역을 확인합니다.

## Path

### GET

/v2/providers/marketplace\_openapi/apis/api/v1/settlement-histories

### Example Endpoint

 https://api-gateway.coupang.com/v2/providers/marketplace\_openapi/apis/api/v1/settlement-histories?revenueRecognitionYearMonth=2019-10

## Request Parameters

### Query String Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| revenueRecognitionYearMonth | | | | | O | String | 매출인식월  형식 : YYYY-MM |

### Request Example

 not require body

### Error Spec

| HTTP 상태 코드(오류 유형) | 오류 메시지 | 해결 방법 |
| --- | --- | --- |
| 400 (요청변수확인) | Invalid revenueRecognitionYearMonth format (yyyy-MM) | 매출인식월을 올바른 형식으로 입력했는지 확인합니다. |
| 400 (요청변수확인) | 해당월까지만 조회할 수 있습니다. | 매출인식월을 당월 이후로 입력했는지 확인합니다. |
