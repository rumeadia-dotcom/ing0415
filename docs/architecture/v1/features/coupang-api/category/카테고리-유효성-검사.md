# 카테고리 유효성 검사

> Zendesk article id: `360033517174`
> 섹션: 카테고리 APIs
> 추출 시점: 2026-05-28

**API 적용 가능한 구매자 사용자 지역: 한국**

해당 노출 카테고리가 현재 사용 가능한지 체크합니다.

카테고리 리뉴얼 등으로 인해, 사용 중인 카테고리가 사용하지 않는 카테고리로 변경될 수 있습니다.  
  
단, 카테고리 리뉴얼은 연 2회 이루어지며 리뉴얼로 인해 변경된 카테고리에 대하여 판매자님들께 공지를 하고 있습니다.

따라서 수시로 카테고리 유효성 검사를 하실 필요는 없습니다.

## Path

GET

/v2/providers/seller\_api/apis/api/v1/marketplace/meta/display-categories/{displayCategoryCode}/status

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/meta/display-categories/{displayCategoryCode}/status

not require body
