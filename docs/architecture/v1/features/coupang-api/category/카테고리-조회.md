# 카테고리 조회

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360034035753>
> Zendesk article id: `360034035753`
> 섹션: 카테고리 APIs
> 추출 시점: 2026-05-28

---

**API 적용 가능한 구매자 사용자 지역: 한국**

카테고리 정보를 노출 카테고리 코드(displayCategoryCode)를 이용하여 조회한다.

1 Depth 카테고리 정보 조회는 노출카테고리코드 값을 0으로 설정 후 호출합니다.

## Path

GET

/v2/providers/seller\_api/apis/api/v1/marketplace/meta/display-categories/{displayCategoryCode}

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/meta/display-categories/0

## Request Parameters

### Path Segment Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| displayCategoryCode | | | | | O | String | 노출카테고리코드 |

### Request Example

not require body

## Response Message

| Name | | | | | Type | Description |
| --- | --- | --- | --- | --- | --- | --- |
| code | | | | | String | 결과코드  SUCCESS/ERROR |
| message | | | | | String | 메세지 |
| data | | | | | Array | 노출카테고리목록  조회된 노출 카테고리 개수만큼 N번 반복 |
|  | displayCategoryCode | | | | String | 노출카테고리코드 |
|  | name | | | | String | 노출카테고리명 |
|  | status | | | | String | 노출카테고리상태  | Status | Description | | --- | --- | | ACTIVE | 서비스중 | | READY | 서비스 준비중 | | DISABLED | 사용 안함 | |
|  | child | | | | String | 하위노출카테고리  조회된 카테고리 코드의 1 Depth 하위 카테고리 List 표시.(2 Depth 이상의 child 는 표시하지 않음.) child 카테고리가 없을 경우 null 또는 빈 배열 반환  1 Depth 카테고리 조회는 displayCategoryCode 파라메터 값을 0으로 설정하면 가능합니다. |
