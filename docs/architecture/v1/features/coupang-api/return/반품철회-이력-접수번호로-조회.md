# 반품철회 이력 접수번호로 조회

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360034027374>
> Zendesk article id: `360034027374`
> 섹션: 반품 APIs
> 추출 시점: 2026-05-28

---

**API** **적용** **가능한** **구매자** **사용자** **지역****:** **한국**

취소(반품)접수번호를 이용하여 철회된 반품의 이력을 조회합니다.

## Path

POST

/v2/providers/openapi/apis/api/v4/vendors/{vendorId}/returnWithdrawList

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/openapi/apis/api/v4/vendors/A00012345/returnWithdrawList

## Request Parameters

### Path Segment Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| vendorId | | | | | O | String | 판매자 ID  쿠팡에서 업체에게 발급한 고유 코드 예) A00012345 |

### Body Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| cancelIds | | | | | O | Array | 철회 이력을 조회할 취소(반품)접수번호 목록  한 번에 최대 50개까지 cancelIds 조회가 가능합니다.  cancelIds는 number 타입이어야 합니다. |

### Request Example

```
{
  "cancelIds": [
    87033689
  ]
}
```

### Error Spec

| --- | --- | --- |
| 500 (요청변수확인) | CReturns getWithdrawnList failed | 올바른 반품접수번호(cancelIds)를 입력했는지 확인합니다. 해당 항목 입력을 안할 경우 500 에러가 발생합니다. |
