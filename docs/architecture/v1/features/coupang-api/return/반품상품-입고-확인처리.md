# 반품상품 입고 확인처리

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360034027214>

**API** **적용** **가능한** **구매자** **사용자** **지역****:** **한국****,** **대만**

반품요청 목록조회 시 receiptStatus가 반품접수(RETURNS\_UNCHECKED)인 반품 건에 대해 처리 가능합니다.

‘빠른환불 대상 상품이 아니거나’, ‘회수 송장이 트랙킹 되지 않는 상품’에 대하여 [반품 상품 입고 확인 처리]를 해주시기 바랍니다.

※ 반품 자동연동 서비스를 이용하시는 경우

- 빠른환불 대상 상품 : 반품 회수 송장 트래킹 되기 때문에 ‘반품 상품 입고 확인 처리’가 불필요 (상품 회수 확인 시 바로 환불됩니다.)

- 빠른환불 미대상 상품 : ‘반품 상품 입고 확인 처리’ 및 ‘반품 요청 승인 처리’필요

[반품 상품 입고 확인 처리]를 하시면, 송장 트래킹이 불가능했던 빠른환불(선환불) 대상 상품은 바로 환불되고, 반품 승인 및 완료 처리됩니다.

따라서 빠른환불 대상 상품이면서, 굿스플로를 사용하는 경우에는 별도의 ‘반품 요청 승인 처리’가 불필요합니다.

- 빠른환불 대상 상품 : 일반 배송이면서, 상품 가격이 10만원 미만인 상품 (신선식품, 주문제작 상품 등은 제외)

  

\*[반품 API Workflow](https://developers.coupangcorp.com/hc/ko/articles/360033643294)를 확인하시면, 보다 쉽게 이해하실 수 있습니다.

\*[반품 상품 입고 확인 처리] 후, 환불이 되었으나 반품된 물건 또는 배송비에 문제가 있을 경우에는 WING에서 ‘쿠팡 확인 요청’을 해주시기 바랍니다.

## Path

PATCH PUT

/v2/providers/openapi/apis/api/v4/vendors/{vendorId}/returnRequests/{receiptId}/receiveConfirmation

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/openapi/apis/api/v4/vendors/A00012345/returnRequests/363585/receiveConfirmation
