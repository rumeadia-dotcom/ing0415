# 교환요청 거부 처리

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360034027874>
> Zendesk article id: `360034027874`
> 섹션: 교환 APIs
> 추출 시점: 2026-05-28

---

**API** **적용** **가능한** **구매자** **사용자** **지역****:** **한국**

고객이 요청한 교환 요청을 수락할 수 없는 경우, 이 API를 사용하여 교환 요청을 거부할 수 있습니다.

거부조건:

1. 교환 상품이 매진된 상태일 경우
2. 고객이 교환 요청을 철회한 경우

## Path

PATCH PUT

/v2/providers/openapi/apis/api/v4/vendors/{vendorId}/exchangeRequests/{exchangeId}/rejection

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/openapi/apis/api/v4/vendors/A00012345/exchangeRequests/100000070/rejection

## Request Parameters

### Path Segment Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| vendorId | | | | | O | String | 판매자 ID  쿠팡에서 업체에게 발급한 고유 코드  예) A00012345 |
| exchangeId | | | | | O | Number | 교환 접수번호  \*교환 요청 목록 조회 API를 이용해서 확인할 수 있습니다. |

### Body Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| exchangeId | | | | | O | Number | 교환 접수번호 |
| vendorId | | | | | O | String | 판매자 ID  쿠팡에서 업체에게 발급한 고유 코드  예) A00012345 |
| exchangeRejectCode | | | | | O | String | 거절원인코드  노트: 교환거절코드는"SOLDOUT" 또는 "WITHDRAW"만 입력할 수 있습니다  * SOLDOUT : 교환할수있지만 아이템이 매진되였습니다. * WITHDRAW :교환요청이 철회되였습니다 (고객님이 요청함) |

### Request Example

```
{
  "exchangeId": "100000070",
  "exchangeRejectCode": "SOLDOUT",
  "vendorId": "A00012345"
}
```
