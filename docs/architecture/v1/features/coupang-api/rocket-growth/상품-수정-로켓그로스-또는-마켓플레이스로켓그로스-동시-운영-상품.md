# 상품 수정 (로켓그로스 또는 마켓플레이스/로켓그로스 동시 운영 상품)

---

**API** **적용** **가능한** **구매자** **사용자** **지역****:** **한국**

업체상품 정보를 수정합니다.  
'상품 조회 API'를 이용하여 조회된 JSON 전문에서 원하는 값 만 수정 후, 전체 JSON 전문을 전송하여 수정이 가능합니다.

> 이 API를 사용하면 승인 후에 반영됩니다.

기 마켓플레이스 상품수정 API를 사용하시는 경우, 동일 Endpoint를 사용하여 로켓그로스 로켓그로스 상품 또는 로켓그로스 & 마켓플레이스 동시 운영 상품에 대한 수정이 가능하지만, 본 문서에 있는 신규 schema를 활용하셔야 합니다. 기존 상품수정 API에서 추가/변경된 parameter는 초록색 텍스트를 참고하실 수 있습니다. 기존 마켓플레이스 상품에 로켓그로스 상품을 추가하시는 경우, 상품수정 API를 활용하여 로켓그로스 상품 관련 parameter 값을 입력하여 상품을 추가하실 수 있습니다.

상품 생성 시 사용한 request body에, [sellerProductId]와 [sellerProductItemId]를 삽입하여 수정 가능  
- **옵션 수정** : 수정하고 싶은 옵션의 아이템 상단에 [sellerProductItemId]와 [vendorItemId] 삽입 후, 원하는 [attributeValueName]으로 수정하여 전송  
- **옵션 삭제** : 삭제하고 싶은 옵션의 'items 배열 하위 값'을 제거하고, 남기고 싶은 아이템 상단에 [sellerProductItemId] 삽입 후 전송 (이미 승인완료된 이력이 있는 옵션은 삭제 불가)  
- **옵션 추가** : [sellerProductItemId]를 입력하지 않고 items 배열을 추가하여 전송

> ※ [sellerProductItemId]는 상품 조회 API를 이용해 확인가능합니다.  
> ※ 승인 완료된 상품의 판매가격, 재고수량, 판매상태, 할인율 기준가는 상품수정API가 아닌, [상품 아이템별 수량/가격/판매여부/할인율기준가 변경] API를 통해 가능합니다.

| 400 | 등록상품명에 허용하지 않는 특수문자가 존재합니다.  노출상품명에 허용하지 않는 특수문자가 존재합니다. | displayProductName 및 sellerProductName    영어, 한글, 숫자, 공백 및 아래 특수 문자만 허용합니다.    !@#$%^&\*()\_+=-][{}';:\"/.,<>?~㈜®|°。× |
| 400 | 중복된 바코드가 존재합니다. | 다른 아이템에 바코드를 중복으로 사용할 수 없습니다.  다른 바코드 또는 자동 생성한 쿠팡 바코드를 입력하세요. |
| 400 | 입고를 생성하신 이후에는 물류(SKU)정보를 변경하실 수 없습니다. | 입고 후에는 SKU 정보 수정이 불가합니다. 필요한 경우 지원팀에 문의하세요. |
| 400 | 누락된 물류정보가 있습니다. 기존 등록된 SKU정보나 변경하실 SKU정보를 입력하여 다시 수정을 요청하세요 | SKU 정보 필드에 일부 값이 누락되었습니다. 해당 필드에 입력해주세요. 또한 SKU 정보는 삭제할 수 없습니다. |
