# 카테고리 메타 정보 조회

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/39429124103449>
> Zendesk article id: `39429124103449`
> 섹션: 로켓그로스 APIs
> 추출 시점: 2026-05-28

---

노출 카테고리코드(displayCategoryCode)를 이용하여 해당 카테고리에 속한 고시정보, 옵션, 구비서류, 인증정보 목록 등을 조회합니다. 상품 생성 시, 쿠팡에서 규정하고 있는 각 카테고리의 메타 정보와 일치하는 항목으로 상품 생성 전문을 구성해야 합니다. 카테고리 메타 정보 조회의 경우 마켓플레이스와 동일한 Endpoint를 사용하며, 조회된 카테고리에 대해 유통기간 정보를 필요 여부에 대한 파라미터가 추가 응답됩니다. 로켓그로스 상품의 경우 해당 파라미터를 기준으로 상품등록 시 유통기간 정보가 제공되어야 합니다.

[\*노출 카테고리코드 조회 방법(Click)](https://developers.coupangcorp.com/hc/ko/articles/360023110213)

[\*필수 구매옵션 및 허용값 확인 방법(Click)](https://developers.coupangcorp.com/hc/ko/article_attachments/37058990801177)

## Example Endpoint

**GET** <https://api-gateway.coupang.com/v2/providers/seller_api/apis/api/v1/marketplace/meta/category-related-metas/display-category-codes/78877>
