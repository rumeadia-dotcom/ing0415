# 카테고리 목록 조회 (로켓그로스 운영 카테고리)

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/39428894927257>
> Zendesk article id: `39428894927257`
> 섹션: 로켓그로스 APIs
> 추출 시점: 2026-05-28

---

**API** **적용** **가능한** **구매자** **사용자** **지역****:** **한국**

노출 카테고리 목록 전체를 조회한다.

[[전체 카테고리 정보 Excel file 다운로드]](https://wing.coupang.com/excel/categories/download/file)

## Path

GET

/v2/providers/seller\_api/apis/api/v1/marketplace/meta/display-categories

### Example Endpoint

GET https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/meta/display-categories?registrationType=RFM&locale=en 

### Query String Parameter

Endpoint URL 상에서 추가 registration type parameter를 입력하여 호출하면 로켓그로스 운영 카테고리 조회가 가능합니다.

|  |  |  |
| --- | --- | --- |
| Parameters | Type | Description |
| registrationType | String | 카테고리의 BusinessType. 등록 유형을 제공하지 않으면 기본적으로 모든 카테고리를 보여줍니다.    입력할 수 있는 값은 RFM과 CGF입니다.  로켓그로스 운영 카테고리의 경우 'RFM'을 입력하여 조회합니다. |
| locale | String | 카테고리 이름의 언어. 사용 가능한 언어는 en, zh-CN, kr입니다. locale을 제공하지 않으면 기본적으로 카테고리 이름은 KR로 표시됩니다. |

### Request Example

not require body

## Response Message

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| Name | | | Type | Description |
| code | | | String | 결과코드  SUCCESS/ERROR |
| message | | | String | 메세지 |
| data | | | Array | 노출카테고리목록  조회된 노출 카테고리 개수만큼 N번 반복 |
|  | displayCategoryCode | | String | 노출카테고리코드  조회된 노출 카테고리 개수만큼 N번 반복 |
|  | name | | String | 노출카테고리명  조회된 노출 카테고리 개수만큼 N번 반복 |
|  | status | | String | 노출카테고리상태  | Status | Description | | --- | --- | | ACTIVE | 서비스중 | | READY | 서비스 준비중 | | DISABLED | 사용 안함 | |
|  | child | | Array | 하위노출카테고리  없으면 [], 카테고리 정보는 동일함. |
|  |  | displayCategoryCode | String | 참조 |
|  |  | name | String | 참조 |
|  |  | status | String | 참조 |
|  |  | child | String | 하위노출카테고리 |
