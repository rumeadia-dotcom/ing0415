**API 적용 가능한 구매자 사용자 지역: 한국**

자동생성옵션을 개별 옵션상품마다 활성화할 수 있는 API 입니다. 판매자님이 등록한 상품에 대해 자동생성옵션을 활성화한다면, 조건에 맞는 옵션이 자동생성 됩니다. 필요 정보는 옵션 ID입니다.

## Path

POST

/v2/providers/seller\_api/apis/api/v1/marketplace/vendor-items/{vendorItemId}/auto-generated/opt-in

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/vendor-items/{vendorItemId}/auto-generated/opt-in

## Request Parameters

### Path Segment Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| vendorItemId | | | | | O | Number | 옵션ID  벤더아이템에 부여되는 고유 번호입니다. |

### Request Example

not require body

## Response Message

| Name | | | | | Type | Description |
| --- | --- | --- | --- | --- | --- | --- |
| code | | | | | String | 결과코드  예) SUCCESS, PROCESSING, FAILED |
| message | | | | | String | 결과 메세지  FAILED 일 경우에 원인을 나타냅니다.  예) System exception |
| data | | | | | String | 결과  예) success |

### Response Example

```
{
    "code": "SUCCESS",
    "message": "",
    "data": "success"
}
```
