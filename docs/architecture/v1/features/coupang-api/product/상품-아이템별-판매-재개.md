# 상품 아이템별 판매 재개

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360033645154>
> Zendesk article id: `360033645154`
> 섹션: 상품 APIs

**API 적용 가능한 구매자 사용자 지역: 한국,대만**

상품 아이템별 판매상태를 판매중으로 변경한다. 이 기능은 판매요청 신청 후 승인완료되어 옵션ID(vendorItemId)가 발급되면 사용할 수 있다.

## Path

PUT

/v2/providers/seller\_api/apis/api/v1/marketplace/vendor-items/{vendorItemId}/sales/resume

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/vendor-items/{vendorItemId}/sales/resume

## Request Parameters

### Path Segment Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| vendorItemId | | | | | O | Number | 옵션ID  벤더아이템에 부여되는 고유 번호입니다. |

### Request Example

not require body

## Response Message

| Name | | | | | Type | Description |
| --- | --- | --- | --- | --- | --- | --- |
| code | | | | | 결과코드 | SUCCESS/ERROR |
| message | | | | | String | 결과 메세지 |

```
{
  "code": "SUCCESS",
  "message": "판매가 재개되었습니다."
}
```

### Error Spec

| HTTP 상태 코드 (오류 유형) | 오류 메시지 | 해결 방법 |
| --- | --- | --- |
| 400 (요청변수확인) | 판매재개에 실패했습니다. [옵션ID[70227\*\*\*178] : 삭제된 상품은 변경이 불가능합니다.] | 옵션ID(vendorItemId)가 삭제 되었는지 확인합니다. |
| 400 (요청변수확인) | 판매재개에 실패했습니다. [옵션ID(55895\*\*\*47)은 쿠팡의 모니터링에 의해 '판매중지'된 상품입니다. | 모니터링에 의해 판매중지된 상품으로 쿠팡 판매자콜센터 또는 온라인문의로 문의하시기 바랍니다. |
| 400 (요청변수확인) | 판매재개에 실패했습니다. [vendoritemid \*\*\* not found] | 옵션ID(vendorItemId) 값을 올바로 입력했는지 확인합니다. |

### URL API Name

RESUME\_PRODUCT\_SALES\_BY\_ITEM
