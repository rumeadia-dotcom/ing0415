# 상품 등록 현황 조회

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/4404525347353>
> Zendesk article id: `4404525347353`
> 섹션: 상품 APIs
> 추출 시점: 2026-05-28

---

**API 적용 가능한 구매자 사용자 지역: 한국，대만**

판매자가 등록할 수 있는 상품수와 현재 등록되어 있는 상품수를 조회한다.

생성 가능한 최대 상품수(permittedCount)가 null일 경우 제한없이 상품등록이 가능합니다.

## Path

GET

/v2/providers/seller\_api/apis/api/v1/marketplace/seller-products/inflow-status

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/seller-products/inflow-status

## Request Parameters

### Request Example

not require body

## Response Message

| Name | | | | | Type | Description |
| --- | --- | --- | --- | --- | --- | --- |
| code | | | | | String | 결과코드  SUCCESS/ERROR |
| message | | | | | String | 결과 메세지 |
| data | | | | |  | 데이터 |
|  | vendorId | | | | String | 판매자 ID  쿠팡에서 업체에게 발급한 고유 코드 |
|  | restricted | | | | Boolean | 상품 생성 불가 여부  false : 생성가능, true : 생성불가능 |
|  | registeredCount | | | | Number | 등록된 상품수(삭제 상품 제외) |
|  | permittedCount | | | | Number | 생성 가능한 최대 상품수  null로 표시될 경우 제한없음 |

### Response Example

```
{
    "code": "SUCCESS",
    "message": "",
    "data": {
      "vendorId": "A00123456",
      "restricted": false,
      "registeredCount": 8125,
      "permittedCount": 10000
    }
}
```
