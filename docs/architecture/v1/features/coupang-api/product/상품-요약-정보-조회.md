# 상품 요약 정보 조회

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360033645094>
> Zendesk article id: `360033645094`
> 섹션: 상품 APIs
> 추출 시점: 2026-05-28

---

**API 적용 가능한 구매자 사용자 지역: 한국，대만**

상품등록/수정 시  입력한 판매자 상품코드(externalVendorSku)로 상품 요약 정보를 조회할 수 있다.

상품 등록이 완료된 후, 반드시 1분 이상 경과한 뒤에 이 API를 호출하여 조회해 주시기 바랍니다.

## Path

GET

/v2/providers/seller\_api/apis/api/v1/marketplace/seller-products/external-vendor-sku-codes/{externalVendorSkuCode}

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/seller-products/external-vendor-sku-codes/{externalVendorSkuCode}

## Request Parameters

### Path Segment Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| externalVendorSkuCode | | | | | O | String | 판매자 상품코드 (업체상품코드) |

### Request Example

not require body

## Response Message

| Name | | | | | Type | Description |
| --- | --- | --- | --- | --- | --- | --- |
| code | | | | | String | 결과코드  SUCCESS/ERROR |
| message | | | | | String | 결과 메세지 |
| data | | | | |  | 업체상품목록  조회된 업체상품 개수만큼 N번 반복 |
|  | sellerProductId | | | | Number | 등록상품ID |
|  | sellerProductName | | | | String | 등록상품명  등록상품명 검색. 20자 이하 |
|  | displayCategoryCode | | | | Number | 노출카테고리코드 |
|  | vendorId | | | | String | 판매자 ID  쿠팡에서 업체에게 발급한 고유 코드 |
|  | saleStartedAt | | | | String | 판매시작일시  "yyyy-MM-dd'T'HH:mm:ss" 형식 |
|  | saleEndedAt | | | | String | 판매종료일시  "yyyy-MM-dd'T'HH:mm:ss" 형식, \*2099년 까지 길게 선택 가능 |
|  | brand | | | | String | 브랜드  브랜드명은 한글/영어 표준이름 입력 |
|  | statusName | | | | String | 업체상품상태명  | Parameter Name | Status | | --- | --- | | IN\_REVIEW | 심사중 | | SAVED | 임시저장 | | APPROVING | 승인대기중 | | APPROVED | 승인완료 | | PARTIAL\_APPROVED | 부분승인완료 | | DENIED | 승인반려 | | DELETED | 상품삭제 | |
|  | createdAt | | | | String | 판매등록일시  "yyyy-MM-dd'T'HH:mm:ss" 형식 |
