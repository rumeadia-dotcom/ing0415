# 상품 조회 (승인불필요)

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360042701211>
> Zendesk article id: `360042701211`
> 섹션: 상품 APIs

---

**API 적용 가능한 구매자 사용자 지역: 한국,대만**

해당 상품의 배송 및 반품지 등의 관련 정보를 조회할 수 있습니다.

본 API의 응답 메시지를 활용하여, [상품 수정 API (승인불필요)]에서 빠르게 정보를 수정할 수 있습니다.

## Path

GET

/v2/providers/seller\_api/apis/api/v1/marketplace/seller-products/{sellerProductId}/partial

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/seller-products/30100201234/partial

## Request Parameters

### Path Segment Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| sellerProductId | | | | | O | String | 등록상품ID  상품 생성 완료 시, 출력 된 등록상품 ID |

### Request Example

not require body
