# 자동생성옵션 비활성화 (옵션 상품 단위)

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/27244841785497>
> Zendesk article id: `27244841785497`
> 섹션: 상품 APIs
> 추출 시점: 2026-05-28

---

**API 적용 가능한 구매자 사용자 지역: 한국**

자동생성옵션을 개별 옵션상품마다 비활성화할 수 있는 API 입니다. 판매자님이 기존 등록 상품에 대해 자동생성옵션을 비활성화 요청한다면, 더 이상 옵션이 자동 생성되지 않습니다. 이미 자동 생성된 옵션을 더 이상 판매하고 싶지 않다면 판매 중지를 설정하시면 됩니다.

## Path

POST

/v2/providers/seller\_api/apis/api/v1/marketplace/vendor-items/{vendorItemId}/auto-generated/opt-out

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/vendor-items/{vendorItemId}/auto-generated/opt-out

## Request Parameters

### Path Segment Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| vendorItemId | | | | | O | Number | 옵션ID  벤더아이템에 부여되는 고유 번호입니다. |

### Request Example

not require body

## Response Message

| Name | | | | | Type | Description |
| --- | --- | --- | --- | --- | --- | --- |
| code | | | | | String | 결과코드  예) SUCCESS, PROCESSING, FAILED |
| message | | | | | String | 결과 메세지  FAILED 일 경우에 원인을 나타냅니다.  예) System exception |
| data | | | | | String | 결과  예) success |

### Response Example

```
{
    "code": "SUCCESS",
    "message": "",
    "data": "success"
}
```

### URL API Name
