# 상품 삭제

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360033644954>
> Zendesk article id: `360033644954`
> 섹션: 상품 APIs
> 추출 시점: 2026-05-28

---

**API 적용 가능한 구매자 사용자 지역: 한국,대만**

등록된 상품을 삭제한다.

상품이 승인대기중 상태가 아니며, 상품에 포함된 옵션(아이템)이 모두 판매중지된 경우에 삭제가 가능합니다.

## Path

DELETE

/v2/providers/seller\_api/apis/api/v1/marketplace/seller-products/{sellerProductId}

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/seller-products/{sellerProductId}

## Request Parameters

### Path Segment Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| sellerProductId | | | | | O | Number | 등록상품ID  상품 생성 완료 시 상품 조회 API를 통해 확인 가능 |

### Request Example

not require body

## Response Message

| Name | | | | | Type | Description |
| --- | --- | --- | --- | --- | --- | --- |
| code | | | | | String | 결과코드  SUCCES/ERROR |
| message | | | | | String | 메세지 |
| data | | | | | String | 삭제된 등록상품ID |

### Response Example

```
{
  "code": "SUCCESS",
  "message": "삭제되었습니다",
  "data": "10***16568"
}
```

### Error Spec

| HTTP 상태 코드 (오류 유형) | 오류 메시지 | 해결 방법 |
| --- | --- | --- |
| 400 (요청변수확인) | 업체상품[103\*\*\*11234]이 없거나 삭제가 불가능한 상태입니다. 삭제는 '저장중', '임시저장' 상태에서만 가능합니다. | 등록상품ID(sellerProductId)이 삭제되었는지 확인합니다. 상품에 포함된 옵션(아이템) 을 먼저 판매중지 처리했는지 확인합니다. |
| 400 (요청변수확인) | 업체상품아이디[None]는 숫자형으로 입력해주세요. | 등록상품ID(sellerProductId) 값을 올바른 값(숫자만 가능)으로 입력했는지 확인합니다. |

### URL API Name

DELETE\_PRODUCT
