자동생성옵션을 전체 상품에 대해 활성화할 수 있는 API 입니다. 자동생성옵션을 활성화한다면, 판매자님의 등록한 상품 중 조건에 맞는 옵션이 자동생성 됩니다.

## Path

POST

/v2/providers/seller\_api/apis/api/v1/marketplace/seller/auto-generated/opt-in

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/seller/auto-generated/opt-in

## Request Parameters

### Request Example

not require body
