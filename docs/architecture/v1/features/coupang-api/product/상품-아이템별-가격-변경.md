# 상품 아이템별 가격 변경

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360034156273>
> Zendesk article id: `360034156273`
> 섹션: 상품 APIs
> 추출 시점: 2026-05-28

---

**API 적용 가능한 구매자 사용자 지역: 한국，대만**

상품 아이템별 판매가격을 변경한다.이 기능은 판매요청 신청 후 승인완료되어 옵션ID(vendorItemId)가 발급되면 사용할 수 있다.

forceSalePriceUpdate=true 으로 가격변경 요청 시 변경 비율 제한없이 가격변경이 가능합니다.

## Path

PUT

/v2/providers/seller\_api/apis/api/v1/marketplace/vendor-items/{vendorItemId}/prices/{price}

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/vendor-items/3572784698/prices/49000?forceSalePriceUpdate=true

## Request Parameters

### Path Segment Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| vendorItemId | | | | | O | Number | 옵션ID  벤더아이템에 부여되는 고유 번호입니다. |
| price | | | | | O | Number | 가격    최소 10원 단위로 입력가능합니다. (1원 단위 입력 불가) |

### Query String Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| forceSalePriceUpdate | | | | |  | Boolean | 가격 변경 비율 제한 여부  false(default 값) or true (가격 변경 비율 제한 없음)  입력 실수 방지를 위해 기존 가격 대비 변경할 수 있는 가격 비율 제한함.  forceSalePriceUpdate=true 를 추가해서 가격변경 요청 시 처리 가능함. |
| **apMinSalePrice** | | | | |  | Number | 자동 가격 조정 최저 판매가. **path의 {price}보다 작아야 함** |
| **apActive** | | | | |  | Boolean | 자동 가격 조정 활성화 여부 (true/false) |

### Error Spec

| --- | --- | --- |
| 400 (요청변수확인) | 가격변경에 실패했습니다. [옵션ID[3572\*\*\*698] : 판매가 변경이 불가능합니다. 변경전 판매가의 최대 50% 인하/최대 100%인상까지 변경가능합니다.] | 가격변경 범위 내의 price 값을 입력했는지 확인합니다. forceSalePriceUpdate=true 를 추가해서 변경 범위 외의 가격을 수정할 수 있습니다. |
| 400 (요청변수확인) | 자동생성옵션의 가격을 직접 수정할 수 없습니다. 기준 판매자옵션 가격으로 자동생성옵션 가격도 변경할 수 있습니다. | 자동생성옵션 생성의 기준이 되는 판매자옵션의 가격을 조정하시면 자동생성옵션의 가격도 수량의 배수만큼 조율 되어 반영 됩니다. 이미 자동생성옵션을 직접 가격 수정한 경우는 판매자옵션 가격으로 조정할 수 없으며, 직접 WING에서 변경하셔야 합니다. |
| 400 (요청변수확인) | 가격변경에 실패했습니다. [옵션ID[5133\*\*\*569] : 삭제된 상품은 변경이 불가능합니다.] | 옵션ID(vendorItemId)가 삭제 되었는지 확인합니다. |
| 400 (요청변수확인) | 가격변경에 실패했습니다. [옵션ID[4685\*\*\*739]: 가격은 최소 10원 단위로 입력가능합니다. (1원단위 입력 불가)] | 가격(price)을 올바른 값으로 입력했는지 확인합니다. 최소 10원 단위로 입력가능합니다. (1원 단위 입력 불가) |
| 400 (요청변수확인) | 유효하지 않은 ID입니다. | 옵션ID(vendorItemId) 값을 올바로 입력했는지 확인합니다. |
| 400 (요청변수확인) | apMinSalePrice must be less than price | apMinSalePrice를 path의 {price}보다 작게 설정하세요 |
| 400 (요청변수확인) | apMinSalePrice and apActive must be provided together | apMinSalePrice / apActive를 함께 전달하세요 |

### URL API Name

UPDATE\_PRODUCT\_PRICE\_BY\_ITEM
