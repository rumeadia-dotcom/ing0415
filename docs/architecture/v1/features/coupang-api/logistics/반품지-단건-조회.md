# 반품지 단건 조회

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360034156013>
> Zendesk article id: `360034156013`
> 섹션: 물류센터 APIs
> 추출 시점: 2026-05-28

---

**API 적용 가능한 구매자 사용자 지역: 한국, 대만**

반품지 센터코드(returncenterCode)로 반품지 정보를 조회합니다.

## Path

GET

/v2/providers/openapi/apis/api/v3/return/shipping-places/center-code

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/openapi/apis/api/v3/return/shipping-places/center-code?returnCenterCodes=1000000051,1000006047

## Request Parameters

### Query String Parameter

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| returnCenterCodes | | | | | O | String | 반품지 센터코드  일괄 조회가능한 returnCenterCodes의  최대 개수는 100.  예) 1000554021,1000554704 |

### Request Example

not require body
