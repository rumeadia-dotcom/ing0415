# 출고지 수정

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/360034155933>
> Zendesk article id: `360034155933`
> 섹션: 물류센터 APIs
> 추출 시점: 2026-05-28

---

**API 적용 가능한 구매자 사용자 지역: 한국, 대만**

출고지를 수정합니다.  
출고지를 수정하려면 outboundShippingPlaceCode 및 remoteInfoId가 필요합니다.  
'[출고지 조회](https://developers.coupangcorp.com/hc/ko/articles/360033644754)' API를 사용하여 outboundShippingPlaceCode 와 remoteInfoId를 얻을 수 있습니다.

※ 출고지 등록 시 설정된 국내 또는 해외의 구분이 결정된 이후에는 국내에서 해외, 해외에서 국내로 변경이 불가합니다. 국내와 해외의 구분을 변경하기 원하실 경우 신규 출고지를 등록해주시기 바랍니다.

## Path

PUT

/v2/providers/openapi/apis/api/v5/vendors/{vendorId}/outboundShippingCenters/{outboundShippingPlaceCode}

### Example Endpoint

https://api-gateway.coupang.com/v2/providers/openapi/apis/api/v5/vendors/A00012345/outboundShippingCenters/123456
