**API 적용 가능한 구매자 사용자 지역: 한국**

이 API는 판매자가 등록 및 등록 완료한 브랜드의 전체 목록을 조회합니다.

## 경로

GET

/v2/providers/seller\_api/apis/api/v1/marketplace/brands/enrolled

### 예시Endpoint

https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/brands/enrolled

## 요청 파라미터

### Body파라미터

없음

### 요청 예시

없음

## 응답 메시지

| Name | | Type | Description |
| --- | --- | --- | --- |
| code | | String | 결과 코드 (SUCCESS / ERROR) |
| message | | String | 결과 메시지 |
| data | | Array<EnrolledBrand> | 등록된 브랜드 목록 |
|  | brandId | String | 브랜드 고유 식별자 (예: KR-5) |
|  | brandName | String | 브랜드 이름 (예: NIKE) |

### 응답 예시

```
{
  "code": "SUCCESS",
  "message": "",
  "data": [
    {
      "brandId": "KR-5",
      "brandName": "NIKE"
    },
    {
      "brandId": "KR-12",
      "brandName": "SAMSUNG"
    }
  ]
}
```

### 오류 사양

| HTTP status code (error type) | Error message | How to resolve |
| --- | --- | --- |
| 401 Unauthorized | Authentication failed | Verify your HMAC authentication credentials |
