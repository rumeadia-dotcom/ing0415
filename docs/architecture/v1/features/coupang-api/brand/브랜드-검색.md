# 브랜드 검색

> 출처: <https://developers.coupangcorp.com/hc/ko/articles/58230017410841>
> Zendesk article id: `58230017410841`
> 섹션: 브랜드 APIs
> 추출 시점: 2026-05-28

---

**사용 가능 구매자 마켓: 한국**

쿠팡 브랜드 라이브러리에서 브랜드명을 기준으로 검색합니다.  
상품 생성 API에서 **brandId**를 입력하기 전에 해당 값을 조회할 때 사용합니다.

## 경로

POST

/v2/providers/seller\_api/apis/api/v1/marketplace/brands/search

### 예시Endpoint

https://api-gateway.coupang.com/v2/providers/seller\_api/apis/api/v1/marketplace/brands/search

## 요청 파라미터

### Body파라미터

| Name | | | | | Required | Type | Description |
| --- | --- | --- | --- | --- | --- | --- | --- |
| brandName | | | | | O | String | 브랜드 검색 키워드 |
| countPerPage | | | | |  | Integer | 페이지당 결과 수 (기본값: 10, 최대: 10) |
| page | | | | |  | Integer | 페이지 번호 (기본값: 1, 최소: 1) |

### 요청 예시

{

"brandName": "NIKE",

"countPerPage": 10,

"page": 1

}

## 응답 메시지

| Name | | | Type | Description |
| --- | --- | --- | --- | --- |
| code | | | String | 결과 코드 (SUCCESS / ERROR) |
| message | | | String | 결과 메시지 |
| data | | | Object | 결과 데이터 |
|  | page | | Integer | 적용된 페이지 번호 |
|  | countPerPage | | Integer | 적용된 페이지 크기 |
|  | totalCount | | Long | 검색 조건에 맞는 총 브랜드 수 |
|  | items | | Array | 브랜드 목록 |
|  |  | brandId | String | 브랜드 고유 ID (예: KR-5) |
|  |  | brandName | String | 브랜드 이름 (예: NIKE) |
|  |  | brandLogoUrl | String | 브랜드 로고 이미지 URL |
|  |  | isUIDRequired | Boolean | UID 제출이 필요한지 여부 |
|  |  | allowedUIDTypes | Array<String> | 허용되는 UID 타입 목록 (예: GTIN, MPN) |

### 응답 예시

```
{
  "code": "SUCCESS",
  "message": "",
  "data": {
    "page": 1,
    "countPerPage": 10,
    "totalCount": 123,
    "items": [
      {
        "brandId": "KR-5",
        "brandName": "NIKE",
        "brandLogoUrl": "https://example.com/logo.png",
        "isUIDRequired": true,
        "allowedUIDTypes": [
          "GTIN",
          "MPN"
        ]
      }
    ]
  }
}
```

### 오류 사양

| HTTP 상태 코드 | 오류 메시지 | 해결 방법 |
| --- | --- | --- |
| 400 Bad Request | brandName is required | brandName 필수 파라미터를 포함하세요 |
| 401 Unauthorized | Authentication failed | HMAC 인증 정보를 확인하세요 |
