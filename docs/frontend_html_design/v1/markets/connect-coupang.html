쿠팡 연결

쿠팡 윙 OpenAPI 키를 입력하여 연결합니다 (외부 리다이렉트 없음)

계정 정보 입력

여러 계정을 구분하기 위한 별명입니다. 1~40자. 동일 마켓에서 중복 불가.

쿠팡 윙 → 마이쇼핑 → 사업자 정보에서 확인. 영문 1자 + 숫자 8자 형식.

입력 즉시 TLS 로 전송 + pgcrypto 암호화 저장됩니다. 브라우저에는 평문이 남지 않습니다.

키 발급 방법

  1. 1. 쿠팡 윙 (wing.coupang.com) 로그인

    사업자 회원 가입이 선행되어야 합니다.

  2. 2. 통합관리 → API 키 발급

    Access Key 와 Secret Key 가 1회 노출됩니다. 즉시 안전한 곳에 복사.

  3. 3. 본 폼에 붙여넣기 → [연결]

    MarketCast 가 HMAC-SHA256 서명으로 매 요청에 자동 부여합니다.

보안 안내

  • 키 입력값은 TLS 로 즉시 전송 + Postgres pgcrypto 로 암호화 저장됩니다.
  • 저장된 키는 브라우저(클라이언트)에 절대 다시 노출되지 않습니다. 마지막 4자만 표시.
  • 모든 외부 호출은 HMAC-SHA256 서명 으로 보호되며, 서명은 Edge Function 내부에서만 생성됩니다.
  • 키 유출이 의심되면 [연결 해제] 후 쿠팡 윙에서 키 재발급하세요.
취소
