여러 계정을 구분하기 위한 별명입니다. 1~40자. 동일 마켓에서 중복 불가.

부여될 권한

  • 상품 등록 · 수정

    MarketCast 에서 작성한 상품을 스마트스토어에 등록하고 가격·재고를 갱신합니다.

  • 상품 목록 조회

    이미 등록된 상품을 조회하여 중복 등록을 방지합니다.

  • 카테고리 트리 조회

    스마트스토어 카테고리를 가져와 매핑에 사용합니다.

보안 안내

  • OAuth 2.0 표준 인증을 사용합니다. 네이버 아이디·비밀번호는 MarketCast 가 받지 않습니다.
  • CSRF 방지를 위한 state 검증을 거칩니다. 인증 요청과 콜백 사이의 위변조를 차단합니다.
  • 발급된 access·refresh 토큰은 암호화 저장되며, 클라이언트(브라우저)에는 절대 노출되지 않습니다.
  • 토큰 만료 임박 시 자동 갱신되며, 셀러가 [연결 해제]를 하면 마켓 측에서도 즉시 폐기됩니다.
