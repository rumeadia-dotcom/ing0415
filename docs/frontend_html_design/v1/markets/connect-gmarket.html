G마켓 연결

ESM 2.0 영구 키를 입력하여 연결합니다 (외부 리다이렉트 없음)

계정 정보 입력

여러 계정을 구분하기 위한 별명입니다. 1~40자.

ESM+ 통합 관리자 계정의 ID. G마켓·옥션 공통.

매 요청 시 JWT (HS256) 서명에 사용됩니다. 입력 후 평문 노출 없음.

G마켓 측 셀러 식별자. ESM+ 에서 확인.

키 발급 방법

  1. 1. ESM+ (sell.esmplus.com) 마스터 계정 로그인

    G마켓·옥션 통합 셀러 관리 포털.

  2. 2. 환경설정 → OpenAPI Secret Key 발급

    발급 즉시 1회 노출. 안전한 곳에 복사 후 본 폼에 붙여넣기.

  3. 3. 셀러 ID 확인 후 [연결]

    MarketCast 가 JWT (sub='sell', aud='sa.esmplus.com', site='G') 를 매 호출에 발급합니다.

보안 안내

  • 키는 TLS 즉시 전송 + pgcrypto 암호화 저장 되며 클라이언트에 다시 노출되지 않습니다.
  • JWT 는 Edge Function 내부에서만 발급 됩니다. 브라우저에는 토큰이 전달되지 않습니다.
  • 옥션도 동일한 마스터 키로 연결할 수 있습니다 (site='A' 로 별도 추가).
